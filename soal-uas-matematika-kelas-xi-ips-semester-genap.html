12. T28soal pas matematikakelas xi (semester genap)tahun pelajaran 2020/2021jawablah!2) buatlah sebuah pernyataan implikasi kemudian tentukan konvers, invers, dankontraposisinya!​


Jawab:

Pernyataan:

"Jika saya bertanya di Brainly, maka saya berusaha belajar dari jawaban penjawab"

Konvers:

"Jika saya berusaha belajar dari jawaban penjawab, maka saya bertanya di Brainly"

Invers:

"Jika saya tidak bertanya di brainly, maka saya tidak berusaha belajar dari jawaban penjawab"

Kontraposisi:

"Jika saya tidak berusaha belajar dari jawaban penjawab, maka saya tidak bertanya di Brainly"

Penjelasan dengan langkah-langkah:

Pernyataan implikasi "jika P, maka Q" ditulis secara matematis sebagai berikut.

Pernyataan: P ⇒ Q

memiliki

Konvers: Q ⇒ P

Invers: -P ⇒ -Q

Kontraposisi: -Q ⇒ -P

Misalkan

P = "Saya bertanya di Brainly"

Q = "Saya berusaha belajar dari jawaban penjawab"

Jadi,

Pernyataan: P ⇒ Q

"Jika saya bertanya di Brainly, maka saya berusaha belajar dari jawabannya"

Sehingga,

Konvers: Q ⇒ P

"Jika saya berusaha belajar dari jawaban penjawab, maka saya bertanya di Brainly"

Invers: -P ⇒ -Q

"Jika saya tidak bertanya di brainly, maka saya tidak berusaha belajar dari jawaban penjawab"

Kontraposisi: -Q ⇒ -P

"Jika saya tidak berusaha belajar dari jawaban penjawab, maka saya tidak bertanya di Brainly"

30. bantuin soal matematika kelas XI


f(x) = (2x + 2)/(x + 8)
y = (2x + 2)/(x + 8)
y(x + 8) = 2x + 2
xy + 8y = 2x + 2
xy - 2x = 2 - 8y
x(y - 2) = 2 - 8y
x = (2 - 8y)/(y - 2)
f⁻¹(x) = (2 - 8x)/(x - 2)
