Jika |a| = b dan b ≥ 0, maka a = b atau a = -b.

a.

|2x| = 10

2x = 10 atau 2x = -10

x = 5 atau x = -5

Jadi, x = -5 atau x = 5

b.

|x| = 5

x = 5 atau x = -5

Jadi, x = -5 atau x = 5

c.

|2x - 5| = 10

2x - 5 = 10 atau 2x - 5 = -10

2x = 15 atau 2x = -5

x = 15/2 atau x = -5/2

Jadi, x = -5/2 atau x = 15/2

d.

|7 - 2x| = 10

7 - 2x = 10 atau 7 - 2x = -10

-2x = 3 atau -2x = -17

x = -3/2 atau x = 17/2

Jadi, x = -3/2 atau x = 17/2

13. Tolong bantu saya mengerjakan soa ini dengan caranya. Terima kasih, soalnya besok mau di kumpulkan


Berdasarkan konsep bangun ruang, kerangka dapat ditentukan melalui :

Kerangka kubus = 12 x rusuk

= 12 x 7cm

= 84cm

Kerangka balok = 4(p + l + t)

= 4(14 + 7 + 7)

= 4(28)

= 112cm

Kerangka limas = 4 x rusuk alas + 4 x sisi tegak

= 4 x 7 + 4 x 7

= 28 + 28cm

= 56cm

Kerangka prisma segitiga sama sisi

= 6 x rusuk segitiga + 3 x tinggi prisma

= 6 x 7 + 3 x 7

= 7(9)

= 63cm

Dijumlahkan seluruhnya

= 84 + 112 + 56 + 63cm

= 315cm

Kawat yang tersedia = 3,5 meter = 350cm

Kawat yang tersisa = 350 - 315cm = 35cm
