1. Pengertian teks eksplanasi

Teks eksplanasi merupakan teks yang menjelaskan sebab akibat suatu fenomena, baik peristiwa alam, ilmu pengetahuan, sosial, budaya, dan lainnya.

2. Struktur teks eksplanasi

•Pernyataan umum

•Deretan penjelas

•Interpretasi

•Simpulan

3. Ciri-ciri teks eksplanasi

•Konteksnya ilmiah

•Penjelasannya logis

•Penjelasannya objektif

•Memaparkan fakta berdasarkan bidang keilmuan

•Menjelaskan proses yang alami, tidak disadari, dan melalui jangka waktu yang panjang

4. Bagian pernyataan umum teks eksplanasi harus dibuat menarik, agar pembaca lebih cepat paham. Karena pada umumnya, pembaca tidak menyukai sesuatu yang basa-basi.

  Js = 12 : [tex]\frac{1}{250.000}[/tex]

  Js = 12 x [tex]\frac{250.000}{1}[/tex]

  Js = 3.000.000 cm : 100.000 = ... km

       = 30 km

2. Jp = S x Js

        = [tex]\frac{1}{1.000.000}[/tex] x 265

        = 0,000265 km x 100.000 = ... cm

        = 26,5 cm

3. S = 1 : [tex]\frac{Js}{Jp}[/tex]

      = 1 : [tex]\frac{320x100.000}{8}[/tex]

      = 1 : [tex]\frac{32.000.000}{8}[/tex]

      = 1 : 4.000.000

4. Js = Jp : S

        = 8 : [tex]\frac{1}{400.000}[/tex]

        = 8 x [tex]\frac{400.000}{1}[/tex]

        = 3.200.000 cm : 100.000 = ... km
