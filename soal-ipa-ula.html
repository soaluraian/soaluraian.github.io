3. test soal matematika saya pemula ​


Jawaban:

f(x) = -x² - 4x

Penjelasan dengan langkah-langkah:

titik puncak P(-2, 4)

p = -2

q = 4

y = a(x - p)² + q

y = a(x - (-2))² + 4

y = a(x + 2)² + 4

melalui titik (0, 0)

x = 0

y = 0

y = a(x + 2)² + 4

0 = a(0 + 2)² + 4

0 = 4a + 4

4a = -4

a = -1

Persamaan kuadrat

y = a(x + 2)² + 4

y = -1(x² + 4x + 4) + 4

y = -x² - 4x - 4 + 4

y = -x² - 4x

f(x) = -x² - 4x
