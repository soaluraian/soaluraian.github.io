Rp135.000,00

B. Rp115.000,00

C.Rp110.000

D. Rp100.000,00

Pembahasan:

Mobil = x dan motor = y

Ditanyakan: 20x + 30y = ....?

Model matematika:

3x + 5y = 17.000  ......(1)

4x + 2y = 18.000  ......(2)

Eliminasi persamaan (1) dan (2) diperoleh:

3x + 5y =17.000 | x4 |12x + 20y = 68.000

4x + 2y =18.000 | x3 |12x + 6y = 54.000 -

                                        ⟺ 14y = 14.000

                                        ⟺ y = 14.000/14

                                        ⟺ y = 1.000

3x+ 5y = 17.000

⟺ 3x + 5(1.000) = 17.000

⟺ 3x + 5.000 = 17.000

⟺ 3x = 17.000 - 5.000

⟺ 3x = 12.000

⟺ x = 12.000/3

⟺ x = 4.000

Jadi, biaya parkir 1 mobil Rp4.000,00  dan 1  motor Rp1.000,00

20x + 30y = 20(4.000) + 30(1.000)

  x +   y = 13 | x4 | 4x + 4y = 52

x + y = 13

⟺ x = 13 -  10

⟺ x = 3

9. Hubungan Ip address dengan gateway


Hubungan Ip address dengan gateway, sebelum membahas lebih lanjut mengenai hubungan IP Address dengan gateway, mari kita bahas secara singkat apa itu IP Adress dan apa itu Gateway. IP Address dapat diartikan sebagai alamat suatu komputer. IP Address berupa kombinasi angka. IP Address sendiri terbagi menjadi beberapa kelas, penggunaan IP Address sendiri berbeda satu dengan yang lainnya. Pemberian IP Address antara satu komputer dengan yang lainnya juga harus berbeda.

Gateway sendiri adalah suatu jalur atau portal yang digunakan komputer pada suatu jaringan komputer, penginputan gateway sendiri juga sama dengan IP Address yaitu kombinasi dari berbagai angka. Adapun hubungan antara IP Address dengan Gateway adalah keduanya saling berkaitan, komputer dengan IP Adress tanpa Gateway tidak akan berfungsi pada jaringan komputer begitu juga sebaliknya. Untuk penggunaan IP Address dan Gateway sendiri sudah terdapat pengaturan yang mengaturnya.

Demikian pembahasan mengenai hubungan antara IP Address dan Gateway, yaitu keduanya saling berkaitan, contoh IP Address komputer A 172.16.253.197 maka Gateway dari komputer tersebut biasanya memiliki angka yang lebih kecil yaitu 172.16.253.195 untuk membaca materi lainnya mengenai IP Address dan materi lainnya mengenai jaringan komputer dapat dibaca pada link berikut

21. tuliskan hubungan antara domain name system (DNS) dengan alamat ip ( IP address)​


Jawaban:

Pengertian IP address

Alamat IP (Internet Protocol Address atau sering disingkat IP) adalah deretan angka biner antar 32-bit sampai 128-bit yang dipakai sebagai alamat identifikasi untuk tiap komputer host dalam jaringan Internet. Panjang dari angka ini adalah 32-bit (untuk IPv4 atau IP versi 4), dan 128-bit (untuk IPv6 atau IP versi 6) yang menunjukkan alamat dari komputer tersebut pada jaringan Internet berbasis TCP/IP.

Pengertian Domain Name System (DNS)

Domain Name System (DNS) adalah distribute database system yang digunakan untuk pencarian nama komputer (name resolution) di jaringan yang mengunakan TCP/IP (Transmission Control Protocol/Internet Protocol). DNS biasa digunakan pada aplikasi yang terhubung ke Internet seperti web browser atau e-mail, dimana DNS membantu memetakan host name sebuah komputer ke IP address.

Penjelasan:

Hubungnnya adalah DNS dan IP adress adalah sama sama sebagai penamaan sistem, bedana jika DNS terdapat nama contohnya (namadomain.com) sedangkan ip dari namadomain.com (192.178.0.1)

MASQUERADE adalah teknik yang digunakan dalam jaringan komputer untuk menyembunyikan alamat IP asli dari host yang terhubung ke internet. 35. hubungan ISP dengan IP Address yaitu...​


Jawaban:

Halo,

Terkait dengan pertanyaanmu "hubungan ISP dengan IP Address yaitu..."

Jika kita ketahui ISP atau singkatan dari (Internet Service Provider) merupakan sebuah layanan yang menyediakan akses internet kepada pelanggan (customer), menyediakan bandwidth dan juga mengatur lalu lintas traffic internet. Lalu, apa hubungannya? ISP menyediakan sebuah jalur khusus untuk mengatur lalu lintas internet diaman disediakan secara random atau acak guna menyambungkan sebuah client ke internet. Hal ini disebut dengan IP Address, atau penomoran IP. ISP menyediakan alamat IP khusus kepada pelanggannya agar tidak ada yang memiliki nomor yang sama dalam mengakses jaringan sehingga tidak menyebabkan terjadinya collision (tabrakan). Ibarat sebuah rumah, pasti kita memiliki jalan masing-masing untuk mencapai rumah kita. Terdapat juga istilah IP Private yang dimana pelanggan menyewa jalan khusus agar ia mendapat alamat IP yang tetap dan tidak berubah.
