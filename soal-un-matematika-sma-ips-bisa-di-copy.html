Rumus Suku ke-n barisan aritmetika 94,

90, 86, 82, ....... adalah? ......

a. Un= 90 + 4n

b. Un= 94 + 4n

c. Un= 94 - 4n

d. Un= 98 - 4n

e. Un= 100 - 6n

Penjelasan dengan langkah-langkah:

aritmetika 94, 90, 86, 82, ...

a = 94

b = -4

Rumus suku ke-n :

Un = a + (n - 1) b

Un = 94 + (n - 1) -4

Un = 94 - 4n + 4

Un = -4n + 98

Un = 98 - 4n

Jawaban:

Un = a + (n-1) b

Un= 94 + 4n

Ket.

a = suku pertama

b = jeda atau beda

n = suku yang ditentukan
