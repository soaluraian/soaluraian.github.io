Sifat-Sifat Bilangan Berpangkat
a. Sifat Perkalian Bilangan Berpangkat
Jika m, n adalah bilangan bulat positif dan a bilangan bulat maka am x an = am + n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 x 33 = 39!
Jawab:
36 x 33 = ( 3 x 3 x 3 x 3 x 3 x 3) x (3 x 3 x 3)
6 faktor 3 faktor
= (3 x 3 x 3 x 3 x 3 x 3 x 3 x 3 x 3)
(6 + 3) faktor
= 26 + 3
= 29

b. Sifat Pembagian Bilangan Berpangkat
Jika m, n bilangan bulat positif dan m > n dengan a bilangan bulat dan a bukan 0, maka am : an = am - n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 : 33 = 33!
Jawab :
36 : 33 = 36
33
= (3 x 3 x 3 x 3 x 3 x 3)
3 x 3 x 3
= (3 x 3 x 3)
(6 - 3) faktor
= 36 - 3
= 33

c. Sifat Perpangkatan Bilangan Berpangkat
Jika a, m, dan n bilangan bulat maka (am)n = am x n.
Contoh Soal:
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (43)2 = 46.
Jawab:
(43)2 = (43) x (43)
= (4 x 4 x 4) x (4 x 4 x 4)
3 faktor 3 faktor
= 4 x 4 x 4 x 4 x 4 x 4
(3 x 2) faktor
= 43 x 2 = 46

d. Sifat Perpangkatan pada Perkalian
Jika a, b bilangan bulat dan m bilangan bulat positifmaka (a x b)m = am x bm.
Contoh Soal=
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (32)3 = 33 x 23.
Jawab:
(32)3 = (3 x 2) x (3 x 2) x (3 x 2)
= (3 x 3 x 3) x (2 x 2 x 2)
3 faktor 3 faktor
= 33 x 23

e. Sifat Perpangkatan pada Pembagian.
Jika a, b bilangan bulat (b bukan 0) dan m bilangan bulat positif, maka (a : b)m = am : bm.
Contoh soal:
Dengan cara menuliskan dalam bentuk pembagian, tunjukkan bahwa (5 : 3)4 = 54 : 34.
Jawab:
(5 : 3)4 = (5/3)4
= (5/3) x (5/3) x (5/3) X (5/3)
= (5 x 5 x 5 x 5) = 54 = 54 : 53
(3 x 3 x 3 x 3) 34

Jawaban soal B berupa esai atau uraian yang ada pada halaman 285 nomor 1 sampai 5 IPS kelas 9 SMP kurikulum 2013 Edisi Revisi 2016 adalah sebagai berikut:

Menurutku, hal-hal yang menjadikan perekonomian Indonesia buruk saat itu adalah Indonesia mendapat beban besar membayar hutang India-Belanda sebagai hasil dari KMB, adanya perubahan sistem ekonomi yang awalnya kolonial menjadi sistem ekonomi nasional, pemerintah mengeluarkan dana berlebih untuk membeli perlengkapan militer yang tidak bersesuaian dengan kondisi ekonomi Indonesia sebagai negara baru.Hal-hal yang mendukung kemajuan perekonomian Indonesia pada masa Orde Baru antara lain adanya tindakan pemerintah menghemat pengeluaran, diterapkannya pajak pendapatan bagi individu atau perseorangan, adanya pembatasan terhadap kredit bank.Penyebab kembali memburuknya perekonomian Indonesia pada tahun 1998 adalah utang luar negeri swasta yang sangat besar dan berjangka pendek memicu kondisi ketidakstabilan ekonomi, sistem perbankan Indonesia memiliki banyak kelemahan, situasi politik yang berkembang buruk dan memanas dipicu terjadinya krisis ekonomi.Perkembangan kehidupan sosial pada bidang pendidikan juga kesejahteraan sejak masa kemerdekaan hingga masa reformasi yakni bahwa pada masa kemerdekaan, kedua aspek tersebut belum diupayakan secara optimal memingat bangsa Indonesia masih berstatus sebagai negara baru. Namun seiring berjalannya waktu hingga masa reformasi, pemerintah berhasil memberdayakan ekonomi masyarakat khususnya dari kecil ke menengah yang turut memperbaiki kesejahteraan hidup bangsa. Selain itu, pemerintah juga senantiasa memperbaiki regulasi dan fasilitas pendidikan agar kesempatan belajar terbuka luas bagi masyarakat. Dahulu misi utama pendidikan adalah memberantas buta huruf lalu kemudian bergeser ke memperbaiki kualitas sumber daya manusia yang merupakan agen pembangunan.Upaya-upaya yang dilaksanakan pemerintah Orde Baru juga pada masa reformasi terkait kekayaan budaya Indonesia adalah memasukkan ke dalam kurikulum pendidikan, diajarkan di berbagai jenjang pendidikan dan juga mendaftarkan hasil kebudayaan asli Indonesia ke pihak UNESCO untuk menghindari munculnya klaim kepemilikan sepihak dari negara lain.» Pembahasan

Kelima soal di atas bentuknya adalah soal esai atau uraian. Materi yang terkait adalah jalannya pemerintahan dari masa kemerdekaan, orde baru hingga ke masa reformasi.
