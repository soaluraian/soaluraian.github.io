8. Disediakan angka 5 6 7 8 dan9 banyak bilangan terdiri dari 3angka berbeda


bentuk permutasi
soal diatas
5 P3 ( 5,3) = 5 ! / (5-3)!
= 5! / 2!
= 5 x 4 x 3 x 2! /2!
= 60 cara

banyaknya cara 3 angka berbeda ada 60 cara

Disediakan angka 5,6,7,8, dan 9.. Maka banyak bilangan yang bisa terbentuk dengan 3 angka yang berbeda.


Jawab :


Karena 3 angka yang berbeda, maka angka yang terbentuk terdiri dari ratusan, puluhan, dan satuan. Dan karena terdiri dari angka yang berbeda maka hilang satu angka setiap kebawah.


•Ratusan = 5,6,7,8,9 = 5

•puluhan = 6,7,8,9 = 4

•satuan = 7,8,9 = 3


Jadi banyak bilangan yang bisa terbentuk adalah 5 x 4 x 3 = 60.

2) f(x) = 3x - 2 dengan daerah asal adalah A = {-2, -1, 0, 1, 2}

a. Daerah hasil atau Range
f(-2) = 3(-2) - 2 = -6 - 2 = -8
f(-1) = 3(-1) - 2 = -3 - 2 = -5
f(0) = 3(0) - 2 = 0 - 2 = -2
f(1) = 3(1) - 2 = 3 - 2 = 1
f(2) = 3(2) - 2 = 6 - 2 = 4
Range = {-8, -5, -2, 1, 4}

b. (-2, -8) => kuadran III
(-1, -5) => kuadran III
(0, -2) => pada sumbu y
(1, 1) => kuadran I
(2, 4) => kuadran I

3) Daerah asal fungsi f dari x ke 4x - 3 adalah {x | -2 < x ≤ 5, x ∈ R}. Tentukan daerah hasilnya.

Karena x ∈ R, maka daerah hasilnya dalam bentuk interval juga.
Misal : y = 4x - 3

-2 < x ≤ 5 => bagaimana caranya si x menjadi 4x - 3 yaitu caranya kita kali 4 lalu dikurang 3

-2 < x ≤ 5 ==> kali 4
-2 . 4 < x . 4 ≤ 5 . 4
-8 < 4x ≤ 20 ====> kurangi 3
-8 - 3 < 4x - 3 ≤ 20 - 3
-11 < 4x - 3 ≤ 17
-11 < y ≤ 17

Jadi daerah hasilnya adalah = {y | -11 < y ≤ 17, y ∈ R}

Jadi
