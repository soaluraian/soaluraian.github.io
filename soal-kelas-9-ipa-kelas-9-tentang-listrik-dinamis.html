5. rumus fisika kelas 9 tentang listrik dinamis


rumus listrik dinamis

Rumus Kuat Arus Listrik (I)

Kuat arus listrik disimbolkan dengan I, memiliki satuan Ampere (A), dirumuskan:

I = Q / t


Keterangan:
I = kuat arus listrik (A)Q = jumlah muatan listrik (Coulomb)t = selang waktu (s)

Rumus beda potensial atau sumber tegangan (V)

Beda potensial atau sumber tegangan disimbolkan dengan V, memiliki satuan Volt (V), dirumuskan:


V = W / Q


Keterangan: 
V = beda potensia atau sumber tegangan listrik (Volt)W = energi (Joule)Q = muatan (Coulomb)

Rumus hambatan listrik (R)

Hambatan atau resistor disimbolkan dengan R, memiliki satuan ohm, dirumuskan:


R = ρ . l / A


Keterangan:
R = hambatan listrik (ohm)ρ = hambatan jenis (ohm.mm2/m)A = luas penampang kawat (m2)

Rumus hukum ohm

Hukum ohm adalah hukum yang menghubungkan antara kuat arus listrik, beda potensial, dan hambatan. Rumus hukum ohm:


I = V / R atau R = V / I, atau V = I . R


Keterangan simbolnya baca pada keterangan simbol rumus sebelumnya.

6. ✨Qyuiz✨Mapel : IPA fisika Kelas : 9Bab : Bab 5 Listrik Dinamis1.kuat arus listrik yang mengalir pada suatu penghantar sebesar 5 A.Berapa muatan listrik yang melewati penghantar itu selama 0,5 menit ?2.Tuliskan beberapa upaya untuk menghemat energi listrik !Not : Tinggal jenius belum TwT​


1. Arus listrik yang mengalir pada suatu penghantar sebesar 5 A selama 0,5 menit memiliki muatan sebesar 150 C.

2. Beberapa upaya untuk menghemat energi listrik antara lain:

Mematikan televisi saat sedang tidak digunakan.Tidak menggunakan kipas angin secara berlebihan.Mencabut kabel charger HP saat sedang tidak mengisi daya.Meminimalisir penggunaan alat alat listrik yang tidak terlalu penting untuk digunakan misalkan hairdryer, music box, perangkat karaoke, dan lain sebagainya.

[tex]~[/tex]

PEMBAHASAN

Listrik dinamis merupakan konsepsi fisika yang berhubungan dengan sifat dari kelistrikan yang dapat bergerak/mengalir. Dalam konsepsi ini ada beberapa hubungan pada masing masing konteksnya misalkan:

1. Hubungan Arus, Muatan, dan Waktu:

[tex]\boxed{\sf I = \frac{q}{t} }[/tex]

Keterangan:

I = Arus listrik (A).q = muatan listrik (Coulumb).t = waktu aliran/pergerakan (sekon).

2. Hubungan Tegangan, Energi, dan Muatan:

[tex]\boxed{\sf V = \frac{W}{q} }[/tex]

Keterangan:

V = Tegangan (Volt).W = Energi listrik (Joule).q = muatan listrik (Coulumb).t = waktu aliran/pergerakan (sekon).

3. Hubungan Arus, Tegangan, dan Hambatan:

[tex]\boxed{\sf I = \frac{V}{R} }[/tex]

Keterangan:

I = Arus listrik (Ampere).V = Tegangan (Volt).R = Hambatan ([tex]\varOmega[/tex]).

[tex]~[/tex]

PENYELESAIAN

Diketahui

I = 5 A.t = 0,5 menit.

[tex]~[/tex]

Ditanyakan

q = ...?

[tex]~[/tex]

Jawab

Gunakan konsepsi Hubungan Arus, Muatan, dan Waktu:

[tex]\rightarrow \sf I = \frac{q}{t} [/tex]

Substitusikan:

[tex]\rightarrow \sf 5 \: A = \frac{q}{0,5 \: menit} [/tex]

Ubah satuan menit ke sekon:

[tex]\rightarrow \sf 5 \: A = \frac{q}{30 \: s} [/tex]

Pindah ruaskan:

[tex]\rightarrow \sf q = 5 \: A \times 30 \: s [/tex]

Kalikan:

[tex]\rightarrow \sf q = 150 \: C [/tex]

[tex]~[/tex]

KESIMPULAN

Jadi, arus listrik yang mengalir pada suatu penghantar sebesar 5 A selama 0,5 menit memiliki muatan sebesar 150 C.

Penyelesaian soal

a. Rangkaian Hambatan 1

[tex]\frac{1}{R_{p} } = \frac{1}{2} + \frac{1}{4} + \frac{1}{6} = \frac{6 + 2 + 3 }{12} = \frac{11}{12}[/tex]

[tex]Rp_{p} = \frac{12}{11}[/tex] = 1,09 Ω

b. Rangkaian Hambatan 2

Rs = 3Ω + 2Ω +4Ω = 9Ω

c. Rangkaian Hambatan 3

Rs = 2Ω + 4Ω = 6Ω

[tex]\frac{1}{R_{p} } = \frac{1}{6} + \frac{1}{6} = \frac{2}{6}[/tex]

[tex]R_{tptal} = \frac{6}{2}[/tex] = 3Ω

d. Rangkaian Hambatan 4

[tex]\frac{1}{R_{p} } = \frac{1}{3} + \frac{1}{6} = \frac{3}{6}[/tex]

[tex]R_{p} = \frac{6}{3}[/tex] = 2Ω

Rtotal = 2Ω + 4Ω = 6Ω

Pembahasan

Pada suatu rangkaian listrik, hambatan listrik juga dapat dipasang  secara seri dan paralel seperti pada lampu dan baterai. Pola pemasangan hambatan  listrik ini ternyata juga memengaruhi besar arus listrik yang mengalir  pada suatu rangkaian listrik.

Rangkaian Hambatan Listrik Seri

Pada rangkaian seri kuat arusnya bernilai sama tetapi tegangannya  berbeda-beda, sehingga:

V1  ≠ V2  ≠ ... ≠ Vn

Vtotal = V1  + V2  + ...+Vn

Menurut hukum Ohm

V1 = I ∙ R1

V2 = I ∙ R2

Vtotal = IR1+ IR2= I (R1  + R2)

Karena :

Vs  = I ∙ Rs   = Vtotal

maka, Rs

Rs  = R1  + R2

Jika ada n buah hambatan yang disusun secara seri maka:

Rs  = R1  + R2  + .... + Rn

Rangkaian Hambatan Listrik Paralel

Paralel  Pada rangkaian paralel,  tegangan listrik bernilai sama  tetapi besar kuat arusnya berbeda,  sehingga:

I1 ≠ I2 ≠ ... ≠ In

Menurut hukum Ohm

[tex]I_{1} = \frac{V}{R_{1} }[/tex] dan [tex]I_{2} = \frac{V}{R_{2} }[/tex]

[tex]I_{total} = I_{1} + I_{2}[/tex]

[tex]I_{total} = \frac{V}{R_{1} } + \frac{V}{R_2 } = V (\frac{1}{R_{1} } + \frac{1}{R_{2 }} )[/tex]

Karena

[tex]I_{total} = \frac{V}{R_{p} }[/tex]

maka,

[tex]\frac{1}{R_{p} } = \frac{1}{R_{1} } + \frac{1}{R_{2} }[/tex]

Sehingga, untuk n buah hambatan paralel:

[tex]\frac{1}{R_{p} } = \frac{1}{R_{1} } + \frac{1}{R_{2} } + ..... + \frac{1}{R_{n} }[/tex]

Jawaban:

4,8 Volt

Penjelasan:

Tentukan R total dalam rangkaian :

Rtotal = R1 + [tex]\frac{R2.R3}{R2 + R3} + R4 + \frac{r1.r2}{r1 + r2}[/tex]

Rtotal = 4 + [tex]\frac{3.3}{3 + 3} + R4 + \frac{1.1}{1 + 1}[/tex]

Rtotal = 4 + 1/2 + 3/2 + 4 = 10 Ohm

Tentukan tegangan listrik pengganti dalam susunan paralel :

Baterai disusun paralel, Jika ε1 = ε2, maka ε = ε1 = ε2 = 12V

Tentukan kuat arus listrik total dalam rangkaian :

Itotal = ε/Rtotal = 12/10 = 1,2 A

Tegangan listrik pada hambatan R4 :

V4 = Itotal×R4 = 1,2×4 = 4,8 Ohm

[tex]R_1=10\,\Omega\\R_2=20\,\Omega\\R_3=6\,\Omega\\R_4=12\,\Omega\\R_5=30\,\Omega[/tex]

[tex]R_{total}[/tex]

R1 dan R2 dipasang secara seri, maka :

[tex]R_{S12}=R_1+R_2\\\\R_{S12}=10\,\Omega+20\,\Omega\\\\R_{S12}=30\,\Omega[/tex]

R3 dan R4 dipasang secara seri, maka :

[tex]R_{S34}=R_3+R_4\\\\R_{S34}=6\,\Omega+12\,\Omega\\\\R_{S34}=18\,\Omega[/tex]

Rs12 dan Rs34 dipasang secara paralel, maka :

[tex]R_P=\frac{R_{S12}\times R_{S34}}{R_{S12}+R_{S34}}\\\\R_P=\frac{30\times18}{30+18}\\\\R_P=\frac{540}{48}\\\\R_P=11,25\,\Omega[/tex]

Rp dan R5 dipasang secara seri, maka :

[tex]R_{total}=R_P+R_5\\\\R_{total}=11,25\,\Omega+30\,\Omega\\\\R_{total}=41,25\,\Omega[/tex]

Jadi, hambatan penggantinya adalah sebesar 41,25 Ω
