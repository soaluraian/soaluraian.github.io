9. hasil dari 18a³ b8 c²/6a³ b³ c6tlong gais buat ph1​


Jawaban:

3b^5 / c⁴

Penjelasan dengan langkah-langkah:

18a³b^8c² / 6a³b³c^6

= 18 : 6 = 3

= a pangkat 3 - 3 = habis

= b pangkat 8 - 3 = 5

= c pangkat 2 - 6 = -4 / pangkat minus akan jadi positif jika jadi penyebut (4)

= 3b^5 / c⁴

12. Nilai rata rata PH 1 Matematika dari 20siswa adalah 72. Jika nilai Unyil tidak diikutkan, maka rata ratanya menjadi 71.Nilai PH1 Matematika milik Unyiladalah...​


72 = 19(71) + 1(y) / (19 + 1)

72 = 1.349 + y / (20)

72 (20) = 1.349 + y

1.440 = 1.349 + y

y = 1.440 - 1.349

y = 91 => Nilai unyil
