1. SOAL FISIKA DASAR MATERI VEKTOR Diketahui Vektor - Vektor Sebagai Berikut A = 32i - 5 34j + 20k B = 2j + 6k Tentukanlah : a. Besaran A dan Besaran B b. 4A - 7B c. -3A + 2B d.(2A) - (-3B) e.(-A) x (4B)


Jawaban:

a. Untuk menghitung besaran (panjang atau magnitudo) vektor A dan vektor B, kita menggunakan rumus:

|Vektor| = √(x^2 + y^2 + z^2)

Untuk vektor A: |A| = √((32)^2 + (-5 34)^2 + (20)^2) = √(1024 + 8500736 + 400)

Untuk vektor B: |B| = √((2)^2 + (6)^2) = √(4 + 36)

b. Untuk menghitung 4A - 7B:

4A = (32 * 4)i + (-5 * 4)34j + (20 * 4)k 4A = 128i - 20 34j + 80k

7B = 14j + 42k

4A - 7B = (128)i - 20 34j + 80k - 14j - 42k 4A - 7B = 128i - 20 34j - 14j + 80k - 42k 4A - 7B = 128i - 20 48j + 38k

c. Untuk menghitung -3A + 2B:

-3A = (-32 * 3)i + (5 * 3)34j - (20 * 3)k -3A = -96i + 15 34j - 60k

2B = 4j + 12k

-3A + 2B = -96i + 15 34j - 60k + 4j + 12k -3A + 2B = -96i + 15 34j + 4j - 60k + 12k -3A + 2B = -96i + 15 38j - 48k

d. Untuk menghitung (2A) - (-3B):

2A = 64i - 10 34j + 40k -3B = -6j - 18k

(2A) - (-3B) = 64i - 10 34j + 40k + 6j + 18k (2A) - (-3B) = 64i - 10 34j + 6j + 40k + 18k (2A) - (-3B) = 64i - 10 28j + 58k

e. Untuk menghitung (-A) x (4B), kita menggunakan metode perkalian silang (cross product) dengan rumus:

Cx = AyBz - AzBy Cy = AzBx - AxBz Cz = AxBy - AyBx

Dimana C = (-A) x (4B)

A = 32i - 5 34j + 20k

A = -32i + 5 34j - 20k

4B = 8j + 24k

Sekarang kita hitung (-A) x (4B):

Cx = (5 34)(24) - (20)(8) = 6808 - 160 Cy = (20)(0) - (-32)(24) = 0 + 768 Cz = (-32)(8) - (5 34)(0) = -256 - 0

Jadi, (-A) x (4B)

1. Sebuah roda menggelinding tanpa slip pada bidang datar. Roda tersebut menempuh jarak 10 meter dalam waktu 5 detik. Jika keliling tersebut 2 meter, maka laju rotasai roda adalah ...... phi rad/s.
Diket :
s = 10 meter
t = 5 sekon
keliling roda = 2πr = 2 meter.....maka r = 2/(2π) = 1/π meter
Ditanya: ω
ω = v / r = (s/t) / r
ω = (10/5) / (1/π)
ω = 2π rad/s

3. Dua buah vektor A dan B masing-masing memiliki besar 3 satuan dan 4 satuan. Kedua vektor membentuk sudut sebesar 30 derajat, maka besar A x B adalah ... satuan.
Diket:
A = 3 satuan
B = 4 satuan
Ф = 30°
Ditanya : A X B
A X B = ABsinФ  = (3)(4) sin30° = 12 .(1/2) = 6 satuan

4. Sebuah benda bergerak sepanjang sumbu x dengan persamaan posisi x(t)=( 18t(kuadrat) –t(pangkat 4) ) m untuk -4 <= t <=4 s. Posisi benda berhenti sesaat adalah ... x= .... m .
Diket :
X(t) = (18t² - t⁴) meter
Ditanya : posisi berhenti (V=0) yg waktunya antara  -4 ≤ t ≤ 4 sekon
jawab:
X(t) = (18t² - t⁴) meter
V(t) = dx/dt = d(18t² - t⁴)/dt
V(t) = 36t - 4t³ = 0
4t ( 9 - t² ) = 0
t= 0 dan t² = 9, t =⁺₋√9 = ⁺₋ 3
t = -3 (tidak memenuhi , krn tak ada waktu mundur atau minus)
t= 3 sekon (memenuhi)
jadi
X(t) =(18t² - t⁴) meter
X(3) =(18(3)² - (3)⁴) meter
X(t) = 81 meter

5. 10 detik setelah bergerak dari keadaan diam, sepeda motor 200 kg bergerak dengan kecepatan 30m/s menuju arah Timur. Impuls sepeda motor selama selang waktu 10 detik tersebut adalah ...... kgm/s ke arah ....
Diket :
t = 10 sekon
m = 200 kg
v = 30 m/s(ketimur)
Ditanya:  I (impuls) dan arahnya
I = F Δt = m Δv
I =  (200 kg) ( 30 m/s) = 2000 kg m s⁻¹
