9. Tolong soal umpn matematika 2017


f(x) = (2x^2 + 3)/(x - 6) ==> f(0) = (2.0^2 + 3)/(0 - 6) = 3/-6 = -1/2
f(x) = u/v
u = 2x^2 + 3 => u' = 4x
v = x - 6 => v' = 1
f'(x) = (u' v - v' u)/v^2
f'(x) = (4x (x - 6) - 1(2x^2 + 3)) / (x - 6)^2
f'(0) = (4(0) (0 - 6) - (2(0)^2 + 3)) / (0 - 6)^2
f'(0) = (0 - 3) / (-6)^2
f'(0) = -3/36 = -1/12

f(0) + f'(0) = -1/2 + (-1/12) = -6/12 - 1/12 = -7/12Gitu ka mungkin aalah soalnya wkwk

≡ Penyelesaian:
[tex]\Rightarrow Q_{pqr}=Q_{pq}+Q_{qr} \\ \Rightarrow Q_{pqr}=[m \times c_{es} \times \Delta T] + [m \times L] \\ \Rightarrow Q_{pqr}=[1\;kg \times 2.100\;J/kg^{o}c \times 5^{o}c]+[1\;kg \times 336.000\;J/kg] \\ \Rightarrow Q_{pqr}=10.500\;Joule+336.000\;Joule \\ \Rightarrow \boxed{Q_{pqr}=346.500\;Joule}[/tex]

37. matematika UNBK trigonometri​


sin(a+b) = sin(a).cos(b)+sin(b).cos(a)

             = -(√(5²-3²))/(5) . (√(13²-12²))/(13) + 12/13 . 3/5

             = -4/5 . 5/13 + 36/65

             = -4/13 + 36/65

             = -20/65 + 36/65

sin(a+b) = 16/65

Jawab:

c

Penjelasan dengan langkah-langkah:

cosα = 3/5 → sinα = 4/5

sinβ = 12/13 → cosβ = -5/13

maka sin(α+β) = sinα.cosβ + cosα.sinβ

                       = -4/5.5/13 + 3/5.12/13

                       = 16/65
