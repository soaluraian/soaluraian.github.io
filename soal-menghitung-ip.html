6. Hitunglah subnetting IP Address Kelas C dari IP 192.10.10.10 a. Jumlah Subnet​


Untuk melakukan subnetting pada alamat IP Kelas C 192.10.10.10, kita akan membagi alamat IP ini menjadi beberapa subnet. Jumlah subnet yang dapat dibuat tergantung pada seberapa banyak bit host yang akan dialokasikan ke dalam subnet dan berapa banyak bit yang akan digunakan untuk subnetting.

Pada alamat IP Kelas C, biasanya, 24 bit pertama (dari bit 0 hingga bit 23) adalah bagian dari alamat jaringan, dan 8 bit terakhir (dari bit 24 hingga bit 31) adalah bagian dari alamat host. Namun, kita dapat membagi alamat IP ini lebih lanjut dengan meminjam beberapa bit dari alamat host untuk digunakan sebagai subnet.

Misalnya, jika kita ingin menggunakan 4 bit untuk subnetting, maka kita akan meminjam 4 bit dari alamat host, sehingga tersisa 4 bit untuk host. Ini akan menghasilkan 2^4 = 16 subnet yang berbeda.

Jadi, jumlah subnet yang dapat dibuat adalah 16.

9. apa itu IP route? bagaimana cara menghitung atau menentukannya?​


Jawaban:

IP route (Internet Protocol route) adalah suatu aturan atau petunjuk yang digunakan dalam jaringan komputer untuk menentukan jalur atau rute yang harus diambil oleh paket data untuk mencapai tujuan yang ditentukan. IP route digunakan dalam protokol jaringan seperti IP (Internet Protocol) untuk mengarahkan paket data melalui jaringan yang kompleks.

Cara menghitung atau menentukan IP route melibatkan beberapa faktor, termasuk topologi jaringan, tabel routing, dan algoritma routing yang digunakan. Berikut adalah langkah-langkah umum untuk menghitung atau menentukan IP route:

1. Identifikasi tujuan: Tentukan alamat tujuan paket data yang ingin dikirim.

2. Periksa tabel routing: Periksa tabel routing pada perangkat jaringan (seperti router) untuk mencari entri yang sesuai dengan alamat tujuan. Tabel routing berisi informasi tentang jaringan yang dapat dijangkau dan jalur yang harus diambil untuk mencapai tujuan tersebut.

3. Pilih jalur terbaik: Jika ada beberapa entri yang sesuai dengan alamat tujuan, gunakan algoritma routing yang digunakan (misalnya, algoritma shortest path) untuk memilih jalur terbaik. Jalur terbaik biasanya ditentukan berdasarkan faktor seperti jarak, kecepatan, atau biaya.

4. Konfigurasi router: Setelah jalur terbaik ditentukan, konfigurasikan router atau perangkat jaringan lainnya untuk mengarahkan paket data melalui jalur tersebut. Ini melibatkan pengaturan tabel routing dan pengiriman paket data ke tujuan melalui jalur yang telah ditentukan.

Penting untuk dicatat bahwa penghitungan atau penentuan IP route dapat lebih kompleks dalam jaringan yang lebih besar dan kompleks. Dalam kasus tersebut, protokol routing dinamis seperti OSPF (Open Shortest Path First) atau BGP (Border Gateway Protocol) digunakan untuk secara otomatis menghitung dan memperbarui tabel routing berdasarkan perubahan dalam jaringan.

18. Cara penghitungan IP subnetting


jumlah subnet, jumlah host per subnet, blok subnet, alamat host dan broadcast yang valid

Jumlah Subnet = 2x, dimana x adalah banyaknya binari 1 pada oktet terakhir subnet mask (2 oktet terakhir untuk kelas B, dan 3 oktet terakhir untuk kelas A). Jadi Jumlah Subnet adalah 22 = 4 subnet

Jumlah Host per Subnet = 2y – 2, dimana y adalah adalah kebalikan dari x yaitu banyaknya binari 0 pada oktet terakhir subnet. Jadi jumlah host per subnet adalah 26 – 2 = 62 host

Blok Subnet = 256 – 192 (nilai oktet terakhir subnet mask) = 64. Subnet berikutnya adalah 64 + 64 = 128, dan 128+64=192. Jadi subnet lengkapnya adalah 0, 64, 128, 192.

Bagaimana dengan alamat host dan broadcast yang valid? Kita langsung buat tabelnya. Sebagai catatan, host pertama adalah 1 angka setelah subnet, dan broadcast adalah 1 angka sebelum subnet berikutnya.

Langkah awal adalah tentukan terlebih dahulu IP adress yang akan di subnetting.Setelah itu gunakan CIDR (Classless Inter-Domain Routing) yang merupakan suatu metode pengalokasian alamat IP dan peruteran IP dalam penulisan IP adress nya.Langkah berikutnya adalah lakukan subnetting terhadap ip private yang termasuk ke dalam kelas C (kelas yang ditentukan dari proses CIDR).Sebagai contoh apabila IP address yang akan kita subnetting adalah 192.168.1.0/24Kita tentukan dulu subnet mask binner = 11111111.11111111.11111111.00000000Berikutnya tentukan subent mask decimal = 255.255.255.0Menghitung jumlah Blok = 2^0 = 1Menghitung jumlah host/blok = 2^8 = 256Diketahui jumlah host valid = 256 – 2 =254Maka diperoleh hasil Blok Pertama: (total host 192.168.1.0 s/d 192.168.1.255)Hasilnya adalah Network address = 192.168.1.0Kemudian ditentukan Host Valid atau IP Valid = 192.168.1.1 s/d 192.168.1.254Hasilnya Broadcast address = 192.168.1.255

Penjelasan:

IP adalah kepanjangan dari Internet Protocol, atau dalam istilah bahasa Indonesia berarti Protokol Internet. IP address atau internet protocol address adalah alamat protokol internet (alamat IP) yang fungsinya untuk mengidentifikasi segala perangkat yang terhubung ke jaringan, baik jaringan internet pada umumnya maupun jaringan internet lokal. Subnetting adalah proses mencuri bit dari bagian HOST dari alamat IP untuk membagi jaringan yang lebih besar menjadi sub-jaringan yang lebih kecil yang disebut subnet. Setelah subnetting, kami berakhir dengan bidang NETWORK SUBNET HOST. Kami selalu mencadangkan  IP adress untuk mengidentifikasi subnet dan satu lagi untuk mengidentifikasi alamat subnet broadcast.

37. Hitung IA, IL, IP, ID, dan IF


Berdasarkan soal tersebut maka diketahui hasil dari indeks harga dengan beberapa metode sebagai berikut :

a. Agregatif sederhana

IA = 92,82%

b. Laspeyres

IL = 106,76%

c. Paasche

IP = 93.68%

d. Drobisch dan Bowley

ID = 100.15%

e. Irving Fisher

IF = [tex]\sqrt{0.998712}[/tex]

Penjelasan dengan langkah-langkah:

Indeks harga dapat diartikan sebagai suatu ukuran yang menunjukkan mengenai berbagai perubahan yang terjadi pada harga dari suatu waktu ke waktu. Indeks harga bertujuan untuk mengukur secara kuantitaif terjadinya perubahan dalam periode yang berlainan.

Diketahui : Data tabel mengenai 5 buah dengan nilai harga dan jumlah barang pada tahun 2012 dan 2013.

Ditanya : Berapa indeks harga tahun 2012 dengan tahun dasar 2013, dengan menggunakan metode:

a. Agregatif sederhana

b. Laspeyres

c. Paasche

d. Drobisch dan Bowley

e. Irving Fisher

Jawab :

Pn = harga pada tahun yang dihitung indeks harganya

Po = harga pada tahun dasar

Qn = kuantitas pada tahun yang dihitung indeks harganya

Qo = kuantitas pada tahun dasar

a. Agregatif sederhana

[tex]IA =[/tex] (∑Pn ÷ ∑Po) × 100%

IA = (22.000 ÷ 23.700) × 100%

IA = 0,92827 × 100%

IA = 92,82%

b. Laspeyres

IL = (∑Pn × ∑Qo) ÷ (∑Po × ∑Qo) × 100%

IL = 10.612.500 ÷ 9.940.000 × 100%

IL = 1,067 × 100%

IL = 106,76%

c. Paasche

IP = (∑Pn × ∑Qn) ÷ (∑Po × ∑Qn) × 100%

IP = 8.121.000 ÷ 8.668.000 × 100%

IP = 0,936 × 100%

IP = 93.68%

d. Drobisch dan Bowley

ID = (IL + IP) ÷ 2 × 100%

ID = (1,067 + 0,936) ÷ 2 × 100%

ID = 1.0015 × 100%

ID = 100.15%

e. Irving Fisher

IF = [tex]\sqrt{IL . IP}[/tex]

IF = [tex]\sqrt{0.998712}[/tex]
