23. SBMPTN 2014 bidang FISIKA.


Oke disini ada asumsi bahwa :
- pada ujung dasar tabung selalu terdapat simpul (bila dihitung dari bawah, gelombang dimulai dengan simpul)
- perut gelombang terjadi pada jarak Ln + ΔL dari dasar tabung

untuk resonasi ke n, dapat dicari panjang seluruh gelombang pada tabung adalah
λn = (1/4 + (1/2(n-1))λ
mis. pada resonasi pertama n = 1 maka panjang seluruh gelombang adalah 1/4λ dihitung dari dasar tabung, dan bertambah dgn kelipatan setengah

kita misalkan total gelombang pada tabung adalah λn dengan jarak tempuhnya adalah Ln + ΔL
Ln + ΔL = λn
λn = (1/4 + (1/2(n-1))λ
Ln + ΔL = (1/4 + (1/2(n-1))λ
kalikan kedua sisi dgn 4
4 (Ln + ΔL) = (1 + 2n - 2)λ
4 (Ln + ΔL) = (2n - 1)λ
maka kita dapat
λ = (4(Ln + ΔL)) / (2n - 1)

ingat v = λf
maka f = v/λ
f = v / ((4(Ln + ΔL)) / (2n - 1))
f = v (2n - 1) / (4(Ln + ΔL))

24. SBMPTN 2014 kode 552


log(2x + 3)/log(1 - |x|) - 1 < 0
[ log(2x + 3) - log(1 - |x|) ] / log(1 - |x|) < 0
[ log(2x + 3)/(1 - |x|) ]/log(1 - |x|) < 0

log((2x + 3)/(1 - |x|)) = 0

10^0 = (2x + 3)/(1 - |x|)
1 = (2x + 3)/(1 - |x|)
1 - |x| = 2x + 3
|x| = -2x - 2
untuk x ≥ 0, maka :
x = -2x - 2
x = -2/3 (tidak memenuhi)
untuk x < 0, maka :
-x = -2x - 2
x = -2

1 - |x| ≠ 0
|x| ≠ 1, x ≠ 1 atau -1

1 - |x| ≠ 1
|x| ≠ 0
x ≠ 0

Jadi, didapatkan batas-batasnya x = 2, x ≠ -1, x ≠ 1, x ≠ 0

Lalu, sebagai syarat tambahan, 1 - |x| > 0 (syarat logaritma tidak boleh negatif)

|x| < 1
-1 < x < 1

Jadi, hanya uji di titik antara -1 dan 1

misal x = -1/2, maka :

(1 - |-1/2|)log(2(-1/2) + 3) = (1/2)log(2) = -1 > 0 (memenuhi)

misal, x = 1/2, maka :
(1 - |1/2|)log(1(1/2) + 3) = (1/2)log(3/2) < 0 (memenuhi)

Jadi, nilai yang memenuhi adalah :

-1 < x < 0 atau 0 < x < 1

====

Adb nilai x = 0 tak memenuhi.

(1 - |0|)log(2(0) + 3) = 1log3

misal 1log3 = x, maka :
1^x = 3 (padahal kita tahu 1 pangkat berapapun hasilnya 1, jadi tidak akan terdefinisi pada x = 0)

36. Soal sbmptn matematika. Mohon pembahasannya


Segitiga BCD dapat dibuat dari vektor BC, CD dan DB, maka :

BC = C - B = (0, y) - (2, y) = (-2, 0)

|BC| = √((-2)² + 0²) = 2

CD = D - C = (0, y/2) - (0, y) = (0, -y/2)

|CD| = √(0² + (-y/2)²) = y/2

DB = B - D = (2, y) - (0, y/2) = (2, y/2)

|DB| = √(2² + (y/2)²) = √(4 + y²/4)


Jadi, panjang segitiga = 2 + y/2 + √(4 + y²/4)


Persegi OABD dapat dibuat dari vektor OA, AB, BD, dan DO, maka :

OA = A - O = (2, 0) - (0, 0) = (2, 0)

|OA| = 2

AB = B - A = (2, y) - (2, 0) = (0, y)

|AB| = y

BD = D - B = (0, y/2) - (2, y) = (-2, y/2)

BD = √(4 + y²/4)

DO = O - D = (0, 0) - (0, y/2) = (0, y/2)

|DO| = y/2

Maka, keliling persegi OABD = 2 + y + √(4 + y²/4) + y/2


Maka, limitnya akan menjadi :

lim y → 2 (2 + y/2 + √(4 + y²/4))/(2 + y + √(4 + y²/4) + y/2)

= (2 + 2/2 + √(4 + 4/4))/(2 + 2 + √(4 + 4/4) + 2/2)

= (2 + 1 + √5)/(4 + 1 + √5)

= (3 + √5)/(5 + √5)

= (3 + √5)/(5 + √5) * (5 - √5)/(5 - √5)

= (15 + 2√5 - 5)(25 - 5)

= (10 + 2√5)/(20)

= (5 + √5)/10
