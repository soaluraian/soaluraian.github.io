38. 25 POINT !MTK kls XIIngasal report!​


Pembahasan

Jika |u| = a, maka

u = - a untuk u < 0

u = a untuk u ≥ 0

Selain itu, kedua ruas dapat dikuadratkan untuk menghilangkan tanda nilai mutlaknya.

Cara Pertama

|2x - 3| + 2 = 9

|2x - 3| = 9 - 2

|2x - 3| = 7

Ada dua kemungkinan:

2x - 3 = -7 atau 2x - 3 = 7

2x = -7 + 3 atau 2x = 7 + 3

2x = -4 atau 2x = 10

x = -2 atau x = 5

Diperoleh HP = {-2, 5}

Cara Kedua (pengkuadratan)

(2x - 3)² = 7²

(2x - 3)(2x - 3) = 7²

4x² - 6x - 6x + 9 = 49

4x² - 12x + 9 = 49

4x² - 12x + 9 - 49 = 0

4x² - 12x - 40 = 0

Sederhanakan dengan kedua ruas dibagi oleh 4.

x² - 3x - 10 = 0

Faktorkan.

(x - 5)(x + 2) = 0

Diperoleh x = 5 atau x = -2

Diperoleh HP = {-2, 5}.
