4. 5. Sari mengikuti Olimpiade Matematika SD. Panitia olimpiade membuat sebuah aturan bahwa setiap jawaban yang dijawab benar diberi nilai 4, jawaban yang salah diberi nilai -1 dan tidak dijawab diberi nilai 0. Apabila dari 50 soal Sari bisa menjawab benar 38 soal dan tidak menjawab 7 soal, maka berapa nilai yang di dapat Sari dalam Olimpiade Matematika SD tersebut? A. 128B. 132 C. 147 D. 145​


Jawaban:

C 147

Penjelasan dengan langkah-langkah:

soal Olimpiade Matematika SD 50

nilai benar 4

salah -1

tidak dijawab 0

benar 38×4 = 152

tidak dijawab 7×0 = 0

salah 50-38-7 = 5

5×(-1)=-5

jumlah nilai 152+0+(-5) = 147

jadi, nilai yang di dapat sari dalam olimpiade matematika SD adalah 147

semoga membantu

6. intan mengikuti Olimpiade Matematika tingkat SD di kotanya ​


Penjelasan dengan langkah-langkah:

34 soal benar

11 soal salah

maka tersisa 5 soal yang tidak dijawab intan

maka nilai intan

34 (2) + 11 (-1) + 5 (0) =

68 - 11 + 0 =

nilai intan = 57

Jawaban:

• Diketahui:

Jawaban benar = skor 2

Jawaban salah = skor -1

Tidak dijawab = skor 0

Soal benar = 34 soal

Soal salah = 11 soal

Jumlah soal = 50

• Ditanya:

Skor yang dimiliki Intan?

• Jawab:

« Jawaban yang tidak dijawab

Jumlah soal = Jawaban benar + jawaban salah – jawaban yang tidak dijawab

=> 50 = 34 + 11 – x

50 = 45 – x

x = 50 – 45

x = 5

« Intan berhasil menjawab

Jawaban benar = 34 × 2 = 68

Jawaban salah = 11 × (-1) = -11

Jawaban yang tidak dijawab = 5 × 0 = 0

« Jumlah skor

= Jawaban benar + jawaban salah + jawaban yang tidak dijawab

= 68 + (-11) + 0

= 57 skor

Jadi, skor yang dimiliki Intan adalah 57 skor


7. mohon bantuannya. soal OSN olimpiade matematika. mohon berikan penjelasan yang sesuai dengan materi SD. terimakasih ​


Bilangan n terbesar yang memenuhi [tex]\frac{1\: +\: 2\: +\: 3\: +\: 4\: + ... +\: n}{n} < 2010[/tex]adalah 4.018.

✦ Pembahasan ✦

Deret Aritmatika merupakan operasi dasar dalam menjumlahkan semua anggota dalam barisan. Dalam deret aritmatika, kita akan menghitung total jumlah dari suatu barisan.

Contoh Soal 

Hitunglah hasil total suku dalam barisan 5, 7, 8, 11, dan 14!

Manual : 5 + 7 + 9 + 11 + 13 = 45

Kita masih bisa menghitung 5 suku tersebut secara manual. Namun, apakah mungkin kita menghitung 100 suku atau lebih dengan cara manual?

Oleh karena itu, kita gunakan rumus :

[tex]\boxed {\bold {Sn = (a + Un) \times \frac{n}{2}}}[/tex]

Namun, dikarenakan rumus :

[tex]\boxed {\bold {Un = a + (n - 1)\times b}}[/tex]

Rumusnya menjadi :

[tex]\boxed {\boxed {\bold {Sn = (2a + (n - 1)\: b) \times \frac{n}{2}}}}[/tex]

Keterangan :

Sn = Jumlah keseluruhan bilangana = Suku pertaman = Suku yang dicarib = Beda atau jarak bilangan

Penyelesaian

[tex]Sn = (2a + (n - 1)\: b) \times \frac{n}{2}[/tex]

[tex]S5 = (2 \times 5 + (5-1) (7-5)) \times \frac{5}{2}[/tex]

[tex]S5 = (10 + 4 (2) ) \times \frac{5}{2}[/tex]

[tex]S5 = (10 + 8 ) \times \frac{5}{2}[/tex]

[tex]S5 = 18 \times \frac{5}{2}[/tex]

[tex]S5 = \frac{90}{2}[/tex]

[tex]\boxed {\boxed {\bold {S5 = 45}}}[/tex]

PenyelesaianDiketahui : [tex]\frac{1\: +\: 2\: +\: 3\: +\: 4\: + ... +\: n}{n} < 2010[/tex]Ditanya : Berapa n terbesar yang mungkin?Jawaban :

Rumus yang hendak digunakan adalah

[tex]\boxed {\bold {Sn = (a + Un) \times \frac{n}{2}}}[/tex]

Namun disini yang ditanya adalah n bukan Sn, sehingga dari persamaan diatas diperoleh :

[tex](1+n) \times \frac{n}{2} < 2010\: n[/tex][tex](1+n)\: n < 2010\: n \times 2[/tex][tex](1+n)\: n < 4020\: n[/tex][tex](1+n) < 4020\: \frac{n}{n}[/tex][tex](1+n) < 4020[/tex][tex]n < 4020 - 1[/tex][tex]n < 4019[/tex]

[tex]\boxed {\boxed {\bold {n = 4018}}}[/tex]  

(Sebab, nilai n harus kurang dari 4019 jadi kemungkinan terbesar adalah 4018).

= 120 - 10 - 5
