13. Persiapan UN 2017 Diutamakan siswa - siswi SMP Gunakanlah cara / langkah yang baik


Mapel : Matematika
Kelas : VIII SMP dan X SMA
Bab : Persamaan Kuadrat

Pembahasan : 
[tex]2x^2 + x - 2 = 0[/tex]

Rumus Akar Persamaan Kuadrat :
X1 + X2 = [tex]- \frac{b}{a} [/tex]
X1 - X2 = [tex] \frac{ \sqrt{D} }{a} [/tex]
X1 . X2 = [tex] \frac{c}{a} [/tex]

Dari Rumus diatas dapat diambil Kesimpulan
p + q = [tex] -\frac{1}{2} [/tex]
pq = [tex]-1[/tex]

Penjumlahan Akar akarnya...
[tex]( \frac{1}{p} + 1) + ( \frac{1}{q} + 1)[/tex]
[tex]2 + [\frac{(p + q)}{pq} ][/tex]
[tex]2 + ( \frac{ \frac{-1}{2} }{-1} )[/tex]
[tex]2 + \frac{1}{2} [/tex]
[tex]= \frac{5}{2} [/tex]

Perkalian Akar akarnya...
[tex]( \frac{1}{p} + 1)( \frac{1}{q} + 1)[/tex]
[tex]1 + \frac{1}{q} + \frac{1}{p} + \frac{1}{pq} [/tex]
[tex]1 + (\frac{p+q}{pq}) + \frac{1}{pq} [/tex]
[tex]1 + [ \frac{ (\frac{-1}{2}) }{-1} ] + \frac{1}{-1} [/tex]
[tex]1 + \frac{1}{2} - 1 [/tex]
[tex]= \frac{1}{2} [/tex]

RUMUS PERSAMAAN KUADRAT BARU...
[tex] x^{2} + [ (\frac{1}{p} + 1) + ( \frac{1}{q} + 1)]x + [( \frac{1}{p} + 1)( \frac{1}{q} + 1)] [/tex] = 0
[tex] x^{2} + ( \frac{-5}{2})x + \frac{1}{2} = 0 [/tex]
[tex]2 x^{2} - 5x + 1 = 0[/tex]

Jadi Persamaan Kuadrat Barunya... [tex]2x^2 - 5x + 1 = 0[/tex]

26. Persiapan UN 2017 Diutamakan siswa - siswi SMP Gunakanlah cara / langkah yang baik


Mapel : Matematika
Kelas : IX SMP
Bab : Barisan dan Deret

Pembahasan :
Barisan Geometri
U1 + U3 = x
[tex]a + ar^2 = x[/tex]

U2 + U4 = y
[tex]ar + ar^3 = y[/tex]

[tex] \frac{x}{y} = \frac{(a + ar^2)}{(ar + ar^3)} [/tex]
[tex] \frac{x}{y} = \frac{a(1 + r^2)}{ar(1 + r^2)} [/tex]
[tex] \frac{x}{y} = \frac{1}{r} [/tex]
[tex] \frac{y}{x} = r[/tex]

U1 + U3 = x
[tex]a(1 + r^2) = x[/tex]
[tex]a = \frac{x}{(1 + r^2)} [/tex]
Substitusi r = y/x
[tex]a = \frac{x}{(1 + \frac{y^2}{x^2}) } [/tex]
[tex]a = \frac{x}{( \frac{x^2 + y^2}{x^2}) } [/tex]
[tex]a = \frac{x^3}{(x^2 + y^2)} [/tex]

Jadi... [tex]U1 = \frac{x^3}{(x^2 + y^2)} [/tex]


U1 + U3 = x
a + ar^2 = x
a(1+r^2) = x...............(1)

U2 + U4 = y
ar + ar^3 = y
ar(1+r^2) = y.............(2)

bandingkan persamaan (1) dan (2)
x/y = a(1+r^2) / ar(1+r^2)
x/y =a/ar
x/y = 1/r
r = y/x.........................(3)

pilih persamaan (1) atau (2)
mis = persamaan (1)
U1 + U3 = x
a(1+r^2) = x
a = x / (1+r^2)

subtitusikan perasamaan (3) pada persamaan (1)
a = x / (1+r^2)
=x / (1+(y/x)^2)
=x / (( x^2 + y^2) / x^2))
= x^3 / (x^2 + y^2)

dengan demikian nilai U1 adalah
U1 = a
= x^3 / (x^2 + y^2)
