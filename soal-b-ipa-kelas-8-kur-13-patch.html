6. perbedaan patch tool dengan brush tool


Jawaban:

Berikut ya kak

Penjelasan:

Patch tool dan brush tool adalah dua alat yang berbeda dalam perangkat lunak pengolah gambar, seperti Adobe Photoshop. Keduanya digunakan untuk mengedit gambar, tetapi memiliki fungsi yang berbeda.

Patch tool digunakan untuk memperbaiki area gambar yang rusak atau tidak diinginkan. Alat ini bekerja dengan cara mengambil sampel area gambar yang sehat dan menggunakannya untuk mengganti area yang rusak.

Brush tool digunakan untuk menggambar atau mewarnai bagian gambar. Alat ini dapat digunakan untuk membuat berbagai efek, seperti menambahkan tekstur atau bayangan.

Perbedaan utama antara patch tool dan brush tool adalah cara kerjanya. Patch tool menggunakan sampel area gambar untuk menggantikan area yang rusak, sedangkan brush tool hanya menambahkan warna atau tekstur ke gambar.
