9. Diketahui 3log 5 = a dan 2log 3 = b. Nilai dari 6log 10 adalah … (Ujian Nasional Matematika Tahun 2013) **Super Intensif UN (Ujian Nasional) SMA/MA IPA , Penerbit: Bisa!Publishing 2016, Penulis: Muhammad Amien & Team


Diketahui:
[tex]^3\log 5=a[/tex] dan [tex]^2\log 3 =b[/tex]
Nilai [tex]^6\log 10 = ...[/tex]

Pembahasan:

[tex]1)\; \log ab=\log a + \log b \\ 2)\; \log \frac{a}{b}=\log a-\log b =-\log \frac{b}{a} \\ 3)\; ^a\log b.^b\log c=^a\log c \\ 4)\; \loga^n=n.\log a \\ 5)\; ^{a^{n}}\log b=^a\log b^{ \frac{1}{n} }= \frac{1}{n}^a\log b \\ 6)\; ^{a^{n}}\log b^{k}= \frac{k}{n}.^a\log b \\ 7)\; a^{^a\log b}=b \\ 8)\; ^a \log b = \frac{^x\log b}{^x \log a}= \frac{1}{^b\log a} \\ 9)^a\log a=1 \\ 10)^a\log 1=0 [/tex]

Dengan menggunakan sifat logaritma diatas, maka diperoleh:
[tex]^6\log 10 \\ = \frac{^3\log 10}{^3\log 6} \\ = \frac{^3\log 2.5}{^3\log 2.3} \\ = \frac{^3\log 2+^3\log 5}{^3\log 2+^3\log 3} \\ = \frac{ \frac{1}{b}+a }{ \frac{1}{b}+1 }\times \frac{b}{b} \\ = \frac{ \frac{b}{b}+ab }{ \frac{b}{b}+b } \\ = \frac{1+ab}{1+b} [/tex]

12. Himpunan penyelesaian dari persamaan cos 2x + cos x = 0 untuk 0˚ ≤ x ≤ 360˚ adalah … (Ujian Nasional Matematika Tahun 2013) **Super Intensif UN (Ujian Nasional) SMA/MA IPA , Penerbit: Bisa!Publishing 2016, Penulis: Muhammad Amien & Team


cos 2x + cos x = 0

⇔ (2 cos² x - 1) + cos x = 0
⇔ 2 cos² x + cos x - 1 = 0
⇔ (2 cos x - 1) (cos x + 1) = 0
⇔ cos x = 1/2 atau cos x = -1

cos x = 1/2 → x = 60°, 300°

cos x = -1 → x = 180°

Jadi, nilai x yang memenuhi adalah {60°, 180°, 300°}

21. SOAL MATEMATIKA 1.Tentukan digit satuan dari 2013+2013²+2013³+2013⁴+...+2013^2013


digit satuan dari suatu bilangan [tex]n[/tex] pada basis [tex]10[/tex] atau desimal, bisa ditulis dalam bentuk

[tex]n\mod 10[/tex]

bisa coba melihat pola dari

[tex]2013^1\mod 10 = 3\\2013^2\mod10 = 9\\2013^3\mod 10 = 7\\2013^4\mod10 = 1\\2013^5\mod 10 = 3\\2013^6\mod10 = 9\\2013^7\mod 10 = 7\\2013^8\mod10 = 1\\\dots[/tex]

polanya berulang

menggunakan sifat [tex](a+b)\mod n = (a\mod n + b\mod n)\mod n[/tex]

didapat

[tex](2013+2013^2+2013^3+2013^4+...+2013^{2013})\mod 10\\= (2013\mod10+\dots+2013^{2013}\mod 10)\mod 10\\= (3+9+7+1+\dots+3)\mod 10\\[/tex]

bilangan yang habis dibagi 4 terakhir adalah 2012, sehingga pola 3+9+7+1 diulang ulang sampai 2012

[tex](3+9+7+1+\dots+3)\mod 10\\=\displaystyle\left(\frac{2012}{4}(3+9+7+1) + 3\right)\mod10\\= (503(20)+3)\mod 10\\= (503\cdot2\cdot10+3)\mod10\\= (503\cdot2\cdot10\mod 10+3\mod 10)\mod10\\= (0+3)\mod 10\\= 3\mod 10\\= 3[/tex]

sehingga digit satuannya adalah [tex]3[/tex]

35. Penyelesaian dari pertidaksamaan 25log (x-3) + 25log (x+1) ≤ ½ adalah … (Ujian Nasional Matematika Tahun 2013) **Super Intensif UN (Ujian Nasional) SMA/MA IPA , Penerbit: Bisa!Publishing 2016, Penulis: Muhammad Amien & Team


25log (x - 3) + 25log (x + 1) ≤ 1/2

⇔ 25log (x - 3)(x + 1) ≤ 25log 25^1/2

⇔ 25log (x² - 2x - 3) ≤ 25log 5

Syarat:

(1) x - 3 > 0 → x > 3
(2) x + 1 > 0 → x > -1
(3) (x² - 2x - 3) ≤ 5

⇔ x² - 2x - 8 ≤ 0

⇔ (x - 4)(x + 2) ≤ 0

⇔ x = 4 atau x = -2

⇔ -2 ≤ x ≤ 4

dari (1), (2), dan (3) diperoleh 3 < x ≤ 4.
