11. apa perbedaan antara web statis dengan web dinamis?


Perbedaan Web Statis dan Web DinamisInteraksi antara pengunjung dan pemilik web
Dalam web statis tidak dimungkinkan terjadinya interaksi antara pengunjung dengan pemilik web. Sementara dalam web dinamis terdapat interaksi antara pengunjung dengan pemilik web seperti memberikan komentar, transaksi online, forum, dll.
Bahasa Script yang digunakan
Web statis hanya menggunakan HTML saja, atau paling tidak bisa ditambah dengan CSS. Sedangkan web dinamis menggunakan bahasa pemrograman web yang lebih kompleks seperti PHP, ASP dan JavaScript.
Penggunaan Database
Web statis tidak menggunakan database karena tidak ada data yang perlu disimpan dan diproses. Sedangkan web dinamis menggunakan database seperti MySQL, Oracle, dll untuk menyimpan dan memroses data.
Konten
Konten dalam web statis hanya diberikan oleh pemilik web dan jarang di-update, sementara konten dalam web dinamis bisa berasal dari pengunjung dan lebih sering di-update. Konten dalam web dinamis bisa diambil dari database sehingga isinya pun bisa berbeda-beda walaupun kita membuka web yang sama.Perbedaan Web Statis dan Web DinamisInteraksi antara pengunjung dan pemilik web

13. bedanya web statis dan web dinamis


Web Statis = merupakan website yang berisikan konten yang sudah fixed (tetap) serta tidak dapat dirubah langsung melalui browser.

Web Dinamis = merupakan website yang berisikan konten yang bisa dirubah kapan saja dan dimana saja. Atau dengan kata lain bisa merubah isi konten website melalui browser langsung.
Web Statis adalah web yang content atau isinya tidak berubah – ubah. Maksudnya adalah isi dari dokumen yang ada di web tersebut tidak dapat diubah secara mudah. Ini dikarenakan karena script yang digunakan untuk membut web statis tidak mendukung untuk mengubah isi dokumen.

Web Dinamis adalah Web yang content atau isinya dapat berubah – ubah setiap saat. Karena dalam teknologi pembuatan web dinamis sudah dirancang semudah mungkin bagi user yang menggunakan web dinamis tersebut.

37. apa perbedaan web statis dan web dinamis ?


Website statis adalah web yang isi kontennya tidak bisa diubah oleh user (kecuali admin). Bahasa yang dipakai biasanya cuma HTML aja dan tidak ada database. Misalnya, website portfolio, website company profile.

Kalau website dinamis, isi kontennya bisa diubah oleh user. Bahasa yang digunakan menggunakan tambahan PHP atau Javascript, ada databasenya. Misalnya, website sosial media, portal berita dll.

Konten apa yang dimaksud? Misalnya, mengganti tampilan foto profil kita, menambah komentar, ini bisa disebut sebagai mengubah konten.

40. konsep pemrograman web terdiri dari..... (soal USBN komputer)


Terdiri dari 2 (dua) macam, yaitu pemrograman web server dengan permograman web client

Pemrograman web server seringkali berhubungan dengan basis data (database), bahasa-bahasa pemrograman web server sangat beragam, mulai dari PHP (Personal Home Page Hypertext Preprocessor), JSP (Java Server Page), ASP (Active Server Pages) dan banyak lagi.

Pemrograman web client adalah bahasa skrip yang digunakan untuk mendukung kelangsungan web server didepan client, bahasa-bahasa pemrograman web client juga beragam, seperti JS (JavaScript) dan VBS (Visual Basic Script)
