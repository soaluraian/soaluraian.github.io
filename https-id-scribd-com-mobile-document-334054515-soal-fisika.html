Jawaban:

Penjelasan:

1. .com

Domain yg Singkatan dari commersial yang artinya komersil. Ekstensi ini di peruntukan bisnis secara umum baik website atau blog perusahaan maupun pribadi

2. .id

Ekstensi domain ini lebih aman digunakan karena dikhususkan untuk website di Indonesia sehingga pemerintah juga dapat turun tangan apabila terjadi kejahatan atau penipuan.

3. .net

Singkatan dari network atau networking dan yang artinya jaringan . nama domain ini diperuntukan untuk nama-nama yang berhubungan dengan jaringan atau perushaan-perusahaan yang bergerak dibidang IT, komputer , layanan-layanan internet dan sejenisnya. Namauun pada umumnya nama domain tersebut bisa diguakan untuk umum, seperti hal nya dengan domain .com. karena dapat didaftarkan dengan tanpa syarat khusus. ".co" domain untuk beberapa negara. ( Jika di Indonesia menggunakan .co.id, namun negara kolombia menggunakan langsung .co )

".id" domain untuk negara Indonesia

".com" domain internasional

".edu" domain untuk pendidikan

".org" domain untuk organisasi

".html" domain untuk tipe ekstensi yang berbentuk html, domain ini juga bisa berupa file dan bisa dijalankan offline

".au" domain untuk negara australia

".onion" domain untuk pengguna aplikasi khusus ( Dalam hal ini butuh aplikasi yang bernama TOR ) terkadang disebut domain untuk dark web

Pembahasan

Yang kita liat diatas ialah top level domain, domain diatas berarti domain yang tidak bisa ditambahkan lagi. Sebagai contoh domain pada web brainly.co.id ialah ".co.id", ".co" adalah second level domain sedangkan ".id" adalah top level domain.

Diatas juga merupakan domain yang bisa digunakan sebagai website dan gmail ( Kecuali .html dan .onion ). Fungsi domain cukup penting sebagai identitas suatu organisasi ataupun website.

1. Sebuah segitiga memiliki tinggi 16 cm dan alas 12 cm, berapa panjang sisi miringnya?

Penjelasan dengan langkah-langkah:

Itu contoh soalnya

Cara jawab:

tinggi (b) = 16 cmalas (a) = 12 cm

Ditanya: miring (c)

Rumus pythagoras

a² + b² = c²

maka

c² = 12² + 16²

c² = 144 + 256

c² = 400

c = √400

c = 20 cm

Jadi sisi miringnya = 20 cm

[tex]c = \sqrt{25} [/tex]
