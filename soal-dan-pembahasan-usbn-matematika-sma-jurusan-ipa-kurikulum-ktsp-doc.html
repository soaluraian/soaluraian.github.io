Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4
= -3/2
= -1 1/2
= -1,5

26. Materi kurikulum 2013 yang jurusan ipa tuh, bagaimana sistem?


Kurikulum 2013 untuk jenjang SMA memakai sistem peminatan dengan tiga pilihan yaitu Matematika dan IPA, IPS, serta Bahasa dan Kebudayaan. Para siswa SMA memilih peminatan sejak duduk di kelas X (I SMA). Seleksi peminatan akan dilakukan berdasarkan nilai rapor SMP dan wawancara oleh guru Bimbingan dan Konseling (BK). 

Peminatan dilihat berdasarkan rapor dan minat anak. Kalau nilai sudah tinggi bisa langsung sesuai minatnya, tapi kalau tidak harus dilihat betul dari wawancara guru BK.

Nilai x yang memenuhi persamaan y = x + 1 dan 2x + y = 10 adalah 3.

Sistem persamaan linear dua variabel adalah suatu sistem di mana terdapat dua atau lebih persamaan linear dengan dua variabel yang memiliki hubungan satu sama lain dan memiliki satu buah penyelesaian.

Bentuk umum sistem persamaan linear dua variabel adalah :

ax + by = c

dx + ey = f

Selain dengan metode substitusi dan eliminasi, ada beberapa langkah tertentu dalam penyelesaian sistem persamaan linear dua variabel, antara lain :

• Mengganti setiap besaran yang ada di masalah tersebut dengan variabel (biasanya dilambangkan dengan huruf atau simbol).

• Membuat model Matematika dari masalah tersebut. Model Matematika ini dirumuskan mengikuti bentuk umum SPLDV.

• Mencari solusi dari model permasalahan tersebut dengan menggunakan metode penyelesaian SPLDV.

Agar lebih jelasnya, simak pembahasan soal berikut.

PEMBAHASAN :

Perhatikan kembali soal di atas.

Diketahui dua persamaan y = x + 1 dan 2x + y = 10 akan dicari nilai x nya. Maka, substitusikan persamaan y ke dalam persamaan kedua.

2x + y = 10

2x + (x + 1) = 10

3x + 1 = 10

3x = 10 - 1

3x = 9

x = 9 ÷ 3

x = 3

Jadi, nilai x yang memenuhi persamaan y = x + 1 dan 2x + y = 10 adalah 3.

35. Dalam tes masuk jurusan ipa di salah satu sma, setiap siswa harus lulus dalam tes matematika dan ipa. dari 180 peserta terdapat 103 orang diinyatakan lulus tes matematika dan 142 siswa dinyatakan lulus tes ipa. banyak siswa yang diterima di jurusan ipa di sma tersebut adalah


Banyak siswa yang diterima di jurusan ipa di SMA adalah 65 peserta

Penjelasan dengan langkah-langkah:

Diketahui:

Himpunan semesta (S) = 180 pesertajumlah peserta lulus tes matematika = 103 pesertajumlah peserta lulus tes ipa = 142 peserta

Ditanya: Banyak siswa yang lulus keduanya sehingga akan diterima di jurusan ipa ...?

Jawab:

S = (103 - x) + x + (142 - x)

180 = 103 + 142 - x

180 = 245 - x

   x = 245 - 180

   x = 65

Jumlah siswa yang lulus tes matematika dan ipa adalah 65 peserta, sehingga banyak siswa yang diterima di jurusan ipa di SMA adalah 65 peserta

Argumen tim afirmasi

Kurikulum Merdeka baik untuk diterapkan karena dapat memfasilitasi para siswa yang memiliki potensi berbeda, dimana tidak ada lagi batasan jurusan yang ditetapkan, sehingga siswa dapat memilih jurusan sesuai minat dan bakat mereka. Argumen tim oposisi

Kurikulum Merdeka kurang tepat untuk diterapkan karena penjurusan yang dihapuskan akan membuat kurikulum yang harus dipersiapkan semakin luas. Selain itu jurusan IPA, IPS dan Bahasa yang ada pada jenjang SMA sudah sangat tepat untuk siswa.

Pembahasan

Argumen merupakan pendapat yang biasanya disampaikan pada saat debat. Argumen yang disampaikan dapat setuju ataupun tidak setuju dengan mosi.
