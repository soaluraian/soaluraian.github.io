[tex] \frac{1}{si} = \frac{1}{20} - \frac{1}{60} = \frac{60}{40} - \frac{20}{40} = \frac{40}{40} [/tex]

dimana:

m = massa(mass)

v = kecepatan(velocity)

Vt = kecepatan akhir(final velocity)

Vo = kecepatan awal(initial velocity)

a = percepatan(acceleration)

t = waktu(time)

h = ketinggian(height)

g = percepatan gravitasi(gravitation acceleration)

Ek = ½mv²

31. soal Fisika kelas X ​


Jawaban:

a. 1A

b. 12Ω

c. V1 = 4A, V2 = 4A, V3 = 4A, dan V4 = 4A

d. ada di lampiran

Penjelasan:

Diketahui:

R1 = R2 = R3 = R4 = 4Ω

Vtot = 12 volt

Rangkaian disusun seri

Karena yang seharusnya dihitung terlebih dahulu adalah resistor gabungan, maka:

b) Besar resistor gabungan

Rtot = R1 + R2 + R3 + R4

Rtot = 4 + 4 + 4 + 4

Rtot = 12Ω

a) Arus listrik

Itot = Vtot/Rtot

Itot = 12/12

Itot = 1A  

Karena pada listrik seri I tot = I1 = I2 = I3 = I4, maka  I1 = I2 = I3 = I4 = 1A

c) Beda potensial

Karena pada listrik seri Vtot = V1 + V2 + V3, sehingga

i) V1 = I1 x Rtot

  V1 = 4 x 1 = 4 volt

ii) V2 = I2 x Rtot

   V2 = 4 x 1 = 4 volt

iii) V3 = I3 x Rtot

    V3 = 4 x 1 = 4 volt

iv) V4 = I4 x Rtot

    V4 = 4 x 1 = 4 volt

d). ada di lampiran
