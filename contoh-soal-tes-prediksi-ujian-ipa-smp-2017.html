Em = Energi Mekanik (J)

28. SOAL PREDIKSI SMMPTN BARAT 2017


Soal prediksi SMMPTN BARAT 2017: Agar pertidaksamaan 4x² + 9x + a² > 9 dipenuhi oleh semua x ∈ R, maka batas batas nilai a adalah a > 3 ¾ atau a < –3 ¾. Hasil tersebut diperoleh dengan menggunakan definit positif. Suatu fungsi kuadrat f(x) = ax² + bx + c dikatakan definit positif (f(x) > 0) jika a > 0 dan D < 0 dengan D adalah diskriminan dan D = b² – 4ac.

Pembahasan

4x² + 9x + a² > 9

4x² + 9x + a² – 9 > 0

a = 4b = 9c = a² – 9  

Syarat Definit positif adalah: a > 0 dan D < 0

1) a > 0 ⇒ 4 > 0 (syarat pertama sudah memenuhi)

2) D < 0

b² – 4ac < 0

9² – 4(4)(a² – 9) < 0

81 – 16a² + 144 < 0

225 – 16a² < 0

(15 – 4a)(15 + 4a) < 0

a = 15/4 atau a = –15/4

a = 3 ¾          a = –3 ¾

dengan menggunakan garis bilangan, diperoleh:

----- (–3 ¾) +++++ (3 ¾) ------

Karena kurang dari nol, ambil daerah yang negatif yaitu:

a < –3 ¾ atau a > 3 ¾

36. pada tes ipa skor untuk jawaban benar adalah 2 jawaban salah adalah -1 dan yang tidak di jawab 0 budi menjawab benar 29 soal dan tidak di jawab 5 soal budi memperoleh skor 52.berapa banyak soal yang diujikan dalam tes ipa itu


Misal:
jawaban benar adalah b = 29 soal
jawaban salah adalah s = ?
tidak dijawab adalah t = 5 soal
cari dulu jumlah soal yg salah
Maka,

52 = b(2) + s(-1) + t(0)
52 = 29(2) + s(-1) + 5(0)
52 = 58 + (-s) + 0
s = 58 - 52
s = 6
Jumlah soal yg salah adalah 6

Jadi, banyak soal adalah 
soal benar + soal salah + tidak dijawab
= 29 + 6 + 5
= 40 soal
