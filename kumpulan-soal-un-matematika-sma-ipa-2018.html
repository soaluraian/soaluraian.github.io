8. find the value of 2021 -2022+2019+2018-2017+2016+2015-2014-......-1​


Jawaban:

We can group the terms that are consecutive integers with opposite signs, as follows:

(2021 - 2022) + (2019 + 2018) + (-2017 + 2016) + (2015 - 2014) + ... + (-2 + 1)

The terms inside parentheses can be simplified as follows:

- 2021 - 2022 = -1

- 2019 + 2018 = 4037

- -2017 + 2016 = -1

- 2015 - 2014 = 1

So the expression becomes:

-1 + 4037 - 1 + 1 - 1 + 1 - 1 + ... - 1

There are 1010 pairs of opposite terms, plus an additional -1 at the end. Therefore, the expression simplifies to:

(-1) * 1010 - 1 = -1021

Therefore, the value of the expression is -1021.

Step-1

y = - 2t² + 120t + 15
15 = - 2t² + 120t + 15
- 2t² + 120t = 0
t² - 60t = 0
t(t - 60) = 0
t = 0 ⇒ kondisi awal
t = 60 ⇒ seharusnya tepat sasaran

Step-2

x = 30t
dengan t = 60 sekon
x = 30(60)

Final Step

Perbedaan jarak ⇒ 1900 - 1800 = 100

⇒ Jatuhnya peluru berjarak 100 m di depan D
