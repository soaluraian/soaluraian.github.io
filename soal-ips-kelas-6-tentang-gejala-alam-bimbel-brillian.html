2. Bila dari 150 orang calon mahasiswa ITB terdapat 75 orang mengikuti bimbel IPA, 65 orang mengikuti bimbel IPS dan 40 orang tidak ikut ipa maupun ips. Maka brp orangkah yg mengikuti bimbel ipa dan ips


IPA = 75
IPA saja = 75 - x
IPS = 65
IPS saja = 65 - x
Kedua duanya (IPA & IPS) = x
Tidak keduanya = 40
Jumlah = 150 orang

=> (75 - x) + (65 - x) + x + 40 = 150
=> 180 - x = 150
=> - x = 150 - 180
=> - x = - 30
=> x = 30

=>Jadi banyak orang yang mengikuti bimbel IPA & IPS = 30 orang

29. Ada 175 siswa dalam satu kelas di bimbel A. Dalam sepekan tiap kelas diwajibkan mempelajari 3 materi, yaitu : IPA, IPS, Bahasa. 19 siswa hanya IPA 36 siswa hanya Bahasa 25 siswa hanya IPS 29 siswa IPS dan bahasa 50 siswa IPA dan bahasa 30 siswa IPA dan IPS 16 siswa tidak belajar ketiganya. Berapa siswa yang belajar ketiganya?


Caranya pakai cara cepat yaitu menggunakan rumus "S+I=L"

S adalah jumlah seluruhnya

I adalah irisan dan hanya boleh diisi yang suka keduanya, ketiganya, dst

L adalah lainnya

S + I = L

175+ I = 19+36+25+29+50+30+16

175 + I =205

I =205-175

I =30 siswa

31. jika a+b=1, b+c=2, dan c+a=3, maka a+b+c=tolong jawab ya soalnya bimbel kelas VII susah​


Jawaban

a + b + c = 3

Penjelasan dengan Langkah-langkah

diketahui :

a + b = 1 kemungkinan a = 1 dan b = 0.

b + c = 2 kemungkinan c = 2

c + a = 3 bisa dipastikan a = 1 dan c = 2

jadi

a = 1 , b = 0 , c = 2

jawab :

a + b + c

1 + 0 + 2 = 3

~Bilangan bulat

a + b = 1

b + c = 2

c + a = 3

a = 1 - 0 = 1

b = 0 - 0 = 0

c = 3 - 1 = 2

Maka :

a + b + c

= 1 + 0 + 2

= 3✔

•••

●Kesimpulan :

a + b + c = 3✔
