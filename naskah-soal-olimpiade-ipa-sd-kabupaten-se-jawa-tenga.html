37. soal olimpiade ipa nomor 19tolong bantuannya


0 < t < ta
• ∆t = ta - 0 = ta
• ∆x = xa - 2 xa = - xa
• v = ∆x / ∆t = - xa / ta

ta < t < 2 ta
• ∆t = 2 ta - ta = ta
• ∆x = xa - xa = 0
• v = ∆x / ∆t = 0 / ta = 0
