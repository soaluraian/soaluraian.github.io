Penjelasan:

Dik :

Cair = 4200 J/kg°C

M = 100 g » kg » 0,1 kg

L = 336000 J/kg

∆T1 = 5°C

∆T2 = 10°C

Dit :

Stot Q-R-S = ?

PQRS

Stot = S1 + S2 + S3

m x c x t + L x m + m x c x t

0,1 x 2100 x 5 + 336000 x 0,1 + 0,1 x 4200 x 10

= 1050 + 33600 + 4200

= 38850 J
