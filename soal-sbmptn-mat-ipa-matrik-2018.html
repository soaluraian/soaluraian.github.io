19. Soal Matematika ipa sbmptn


(a - |a - 2|)/a > 2
(a - |a - 2|)/a - 2 > 0
(a - |a - 2|) - (2a))/a > 0
(-a - |a - 2|)/a > 0

tentukan batas-batasnya terlebih dahulu.

untuk a - 2 ≥ 0, a ≥ 2

(-a - (a - 2))/a > 0
(-2a + 2)/a > 0

-2a + 2 = 0
a = 1

Uji pada titik a = 3, maka

(-2(a) + 2)/a = (-2(3) + 2)/3 = (-6 + 2)/3 = -4/3 < 0
Jadi, tidak ada bilangan yang memenuhi

untuk a - 2 < 0, a < 2

(-a - (-(a - 2))/a > 0
(a - (-a + 2))/a > 0
(a + a - 2)/a > 0
(2a - 2)/a > 0

2a - 2 = 0
a = 1
a ≠ 0 (syarat penyebut)

Uji pada titik a = -1, a = 1/2

(2(-1) - 2)/(-1) = (-2 - 2)/(-1) = 4 > 0 (memenuhi, jadi a < -1)
(2(1/2) - 2)/(1/2) = (1 - 2)/(1/2) = -2 < 0 (tidak memenuhi)

Jadi, penyelesaiannya a = {-9, -8, -7, -6, -5, -4, -3, -2, -1}

-9 - 8 - 7 - 6 - 5 - 4 - 3 - 2 - 1 = -45

24. ada yang taukah tentang sbmptn 2018 ini??


Ada, sistem nilai untuk SBMPTN 2018 ini berbeda dengan tahun sebelumnya, dulu kalau jawabannya benar +4, gak dijawab 0, dan kalau salah -1. Kalau sekarang kalau jawabannya benar +1, gak dijawab 0, kalau salah -1, penggantian sistem nilai tersebut agar siswa dalam mengerjakan soal SBMPTN tidak asal-asalan (ngawur).
