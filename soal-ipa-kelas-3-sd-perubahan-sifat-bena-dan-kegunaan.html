Jawab:

1. 12³ + 7³ = ....

= (12×12×12) + (7×7×7)

= 1.728 + 343

= 2.071

2. 21³ - 16³ = ....

= (21×21×21) - (16×16×16)

= 9.261 - 4.096

= 5.165

3. 16³ × 3³ = ....

= (16×16×16) × (3×3×3)

= 4.096 × 27

= 110.592

4. 25³ ÷ 12³ = ...

= (25×25×25) ÷ (12×12×12)

= 15.625 ÷ 1.728

= 9,042

5. (⅓ × 12)³ × 2³ + 4³ ÷ 2³

= ((⅓ × 12)³ × 2³) + (4³ ÷ 2³)

= ((4)³ × 2³) + (2⁶ ÷ 2³)

= (64 × 8) + (2⁶-³)

= 512 + 2³

= 512 + 8

= 520
