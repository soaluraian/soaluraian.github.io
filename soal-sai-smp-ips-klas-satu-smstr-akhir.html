11b + a = 35

——————— (-)

8b = 24

b = 3

Cari nilai a,

3b + a = 11

3(3) + a = 11

9 + a = 11

a = 2

Suku ke-20,

= a + 19b

= 2 + 19(3)

= 2 + 57

= 59

Jawab:

C. 59

U4 = 11, U12 = 35.

U20?

Cari beda (b)

b = (U12-U4)/(12-4)

b = (35-11)/(12-4)

b = 24/8

b = 3

Cari suku pertama (a)

b = (U12-a)/(12-1)

U12-a = b(12-1)

a = U12 - b(12-1)

a = 35 - 3(12-1)

a = 35 - 36 + 3

a = 2

Cari U20 dengan rumus

Un = a+(n-1)b

U20 = 2+60-3

59

35. matematika klas 7 smp


Mapel = Matematika
Bab = Bangun Datar
Materi = Segitiga
------------------------------------------------- ✌

Luas Segitiga DEF
= ½ x a x t
= ½ x 14 cm x 12 cm
= 7 cm x 12 cm
= 84 cm²

EF = √(12 cm)² + (5 cm)²
EF = √144 cm² + 25 cm²
EF = √169 cm²
EF = 13 cm

Keliling Segitiga DEF
= DE + EF + DF
= 14 cm + 13 cm + 21 cm
= 27 cm + 21 cm
= 48 cm
