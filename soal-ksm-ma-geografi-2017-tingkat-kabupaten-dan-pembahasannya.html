d1 = r = (1/2)s

[tex]d2 = \sqrt{ {( \frac{1}{4}s )}^{2} + {( \frac{1}{4} s)}^{2} } [/tex]

[tex] = \sqrt{ \frac{1}{16} {s}^{2} + \frac{1}{16} {s}^{2} } = \sqrt{ \frac{1}{8} {s}^{2} } \\ = \frac{s}{2 \sqrt{2} } = \frac{s}{4} \sqrt{2} [/tex]

[tex]luas = {s}^{2} - \pi {r}^{2} + 4 \frac{d1 \times d2}{2} [/tex]

[tex]326 = {s}^{2} - \pi {( \frac{1}{2} s)}^{2} + 2( \frac{1}{2} s)( \frac{s}{4} \sqrt{2} )[/tex]

[tex]326 = {s}^{2} - \pi {( \frac{1}{4} s ^{2} )} + ( s)( \frac{s}{4} \sqrt{2} )[/tex]

[tex]326 = {s}^{2} - { \frac{\pi}{4} s ^{2} } + \frac{s ^{2} }{4} \sqrt{2}[/tex]

[tex]326 = {s}^{2} (1 - \frac{\pi}{4} + \frac{ \sqrt{2} }{4} )[/tex]

28. Quiz!!Naskah soal Seleksi KSM tingkat kabupaten /kota Bidang matematika Terintegrasi nomor 16Pake jalan and no ngasal​


143/23 = 6 5/23 , h= 6

5/23 = 1/(23/5)

23/5 = 4 3/5 , i = 4

3/5 = 1/(5/3)

5/3 = 4 2/3 , j = 1 , k = 2, l = 3

h = 6, i = 4, j = 1, k = 2, l = 3

kˡ + i×h-j =

2³ + 4×6 - 1 =

8 +24 -1 =

31

jawaban :

kˡ + i×h-j = 31

[tex]\frac{143}{23}=\frac{138}{23}+\frac{5}{23}\\\\\frac{143}{23}=6+\frac{5}{23}[/tex]
h = 6
lalu,
[tex]\frac{5}{23}=\frac{1}{x}\\\\x=\frac{23}{5}\\\\x=\frac{20}{5}+\frac{3}{5}\\\\x=4+\frac{3}{5}[/tex]
jika
[tex]x=i+\frac{1}{j+\frac{k}{l}}\\\\4+\frac{3}{5}=i+\frac{1}{j+\frac{k}{l}}[/tex]
i = 4
dan
[tex]\frac{3}{5}=\frac{1}{j+\frac{k}{l}}\\\\j+\frac{k}{l}=\frac{5}{3}\\\\j+\frac{k}{l}=\frac{3}{3}+\frac{2}{3}\\\\j+\frac{k}{l}=1+\frac{2}{3}[/tex]
j = 1,
k = 2,
l = 3
maka dari itu
nilai kˡ + (i×h) - j
= 2³ + (4×6) - 1
= 8 + 24 - 1
= 32 - 1
= 31
