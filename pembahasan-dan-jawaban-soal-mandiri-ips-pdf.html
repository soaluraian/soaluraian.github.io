Tugas Mandiri

Hitung Luas permukaan dan Volume Prisma segitiga berikut ini :

1. Luas permukaan = 220 + 120[tex]\sqrt{5}[/tex] cm²

Volume = 500 cm³

2. Luas permukaan = 396cm²

Volume = 432cm³

3. Luas permukaan = 1.152cm²

Volume = 1.920cm³

4. Luas permukaan = 888cm²

Volume = 1.056cm³

5. Luas permukaan = 360cm²

Volume = 300cm³

Rumus Mencari Luas permukaan dan Volume prisma segitiga :

Luas permukaan = 2xLuas segitiga + 3xLuas persegi panjang

Volume = [tex]\frac{1}{2}[/TEX] x alas segitiga x tinggi segitiga x tinggi prisma

Pembahasan :

1. Diketahui :

alas segitiga = 10cm

tinggi segitiga = 10cm

tinggi prisma = 12cm

Ditanya :

Luas permukaan ?

Volume prisma segitiga ?

Dijawab :

Kita cari dahulu sisi segitiga yang lainnya, karena ini segitiga sama kaki maka sisi yg lainnya sama besar

sisi = [tex]\sqrt{(\frac{1}{2} \times 10)^2 + 10^2}[/tex]

sisi = [tex]\sqrt{25 + 100}[/tex]

sisi = [tex]\sqrt{125}[/tex]

sisi = 5[tex]\sqrt{5}[/tex] cm

Luas permukaan = 2xLuas segitiga + 3xLuas persegi panjang

= 2([tex]\frac{1}{2}[/tex] x 10 x 10) + (10 x 12) + 2(5[tex]\sqrt{5}[/tex] x 12)

= 100 + 120 + 120[tex]\sqrt{5}[/tex]

= 220 + 120[tex]\sqrt{5}[/tex] cm²

Volume = [tex]\frac{1}{2}[/TEX] x alas segitiga x tinggi segitiga x tinggi prisma

= [tex]\frac{1}{2}[/TEX] x 10 x 10 x 12

= 500 cm³


2. Diketahui :

alas segitiga = 9cm

tinggi segitiga = 12cm

tinggi prisma = 8cm

Ditanya :

Luas permukaan ?

Volume prisma segitiga ?

Dijawab :

Kita cari dahulu sisi segitiga yang lainnya (segitiga siku-siku)

sisi miring = [tex]\sqrt{9^2 + 12^2}[/tex]

sisi miring = [tex]\sqrt{81 + 144}[/tex]

sisi miring = [tex]\sqrt{225}[/tex]

sisi miring = 15cm

Luas permukaan = 2xLuas segitiga + 3xLuas persegi panjang

= 2([tex]\frac{1}{2}[/tex] x 9 x 12) + (9 x 8) + (12 x 8) + (15 x 8)

= 108 + 72 + 96 + 120

= 396cm²

Volume = [tex]\frac{1}{2}[/TEX] x alas segitiga x tinggi segitiga x tinggi prisma

= [tex]\frac{1}{2}[/TEX] x 9 x 12 x 8

= 432cm³


3. Diketahui :

alas segitiga = 12cm

tinggi segitiga = 16cm

tinggi prisma = 20cm

Ditanya :

Luas permukaan ?

Volume prisma segitiga ?

Dijawab :

Kita cari dahulu sisi segitiga yang lainnya (kita anggap segitiga siku-siku)

sisi miring = [tex]\sqrt{16^2 + 12^2}[/tex]

sisi miring = [tex]\sqrt{256 + 144}[/tex]

sisi miring = [tex]\sqrt{400}[/tex]

sisi miring = 20cm

Luas permukaan = 2xLuas segitiga + 3xLuas persegi panjang

= 2([tex]\frac{1}{2}[/tex] x 12 x 16) + (12 x 20) + (16 x 20) + (20 x 20)

= 192 + 240 + 320 + 400

= 1.152cm²

Volume = [tex]\frac{1}{2}[/TEX] x alas segitiga x tinggi segitiga x tinggi prisma

= [tex]\frac{1}{2}[/TEX] x 12 x 16 x 20

= 1.920cm³


4. Diketahui :

alas segitiga = 16cm

tinggi segitiga = 6cm

tinggi prisma = 22cm

Ditanya :

Luas permukaan ?

Volume prisma segitiga ?

Dijawab :

Kita cari dahulu sisi segitiga yang lainnya, karena ini segitiga sama kaki maka sisi yg lainnya sama besar

sisi = [tex]\sqrt{(\frac{1}{2} \times 16)^2 + 6^2}[/tex]

sisi = [tex]\sqrt{64 + 36}[/tex]

sisi = [tex]\sqrt{100}[/tex]

sisi = 10cm

Luas permukaan = 2xLuas segitiga + 3xLuas persegi panjang

= 2([tex]\frac{1}{2}[/tex] x 16 x 6) + (16 x 22) + 2(10 x 22)

= 96 + 352 + 440

= 888 cm²

Volume = [tex]\frac{1}{2}[/TEX] x alas segitiga x tinggi segitiga x tinggi prisma

= [tex]\frac{1}{2}[/TEX] x 16 x 6 x 22

= 1.056 cm³


5. Diketahui :

alas segitiga = 5cm

tinggi segitiga = 12cm

tinggi prisma = 10cm

Ditanya :

Luas permukaan ?

Volume prisma segitiga ?

Dijawab :

Kita cari dahulu sisi segitiga yang lainnya (segitiga siku-siku)

sisi miring = [tex]\sqrt{5^2 + 12^2}[/tex]

sisi miring = [tex]\sqrt{25 + 144}[/tex]

sisi miring = [tex]\sqrt{169}[/tex]

sisi miring = 13cm

Luas permukaan = 2xLuas segitiga + 3xLuas persegi panjang

= 2([tex]\frac{1}{2}[/tex] x 5 x 12) + (5 x 10) + (12 x 10) + (13 x 10)

= 60 + 50 + 120 + 130

= 360cm²

Volume = [tex]\frac{1}{2}[/TEX] x alas segitiga x tinggi segitiga x tinggi prisma

= [tex]\frac{1}{2}[/TEX] x 5 x 12 x 10

= 300cm³

14. misi mau nanya bagaimana cara membuat PDF dengan menggunakan WPS oofficesoalnya tugasnya disuruh buat cerpen nanti jawabannya dikirim menggunakan PDF


1. Mulai buat dokumen baru
Lalu buka aplikasi WPS Office yang sudah terpasang. Pada halaman muka kamu tinggal ketuk ikon "+". Kemudian kamu pilih opsi Dokumen.

2. Pilih jenis template dokumen
Dalam membuat dokumen baru di WPS Office, kamu bisa memilih tampilan kosong atau memilih berbagai template yang disediakan di dalamnya.

3. Mulai membuat PDF dari WPS Office
Lakukan pengeditan sesuai dengan kebutuhanmu dan untuk mulai menyimpan file, kamu tinggal ketuk ikon Save yang ada di pojok kiri atas.
Pada tampilan selanjutnya, kamu tinggal ketuk bagian Jalur untuk menyimpan file PDF di memori internal HP.

4. Pilih lokasi penyimpanan
Selanjutnya kamu tinggal pilih tempat akan menyimpan file, pilih Lokasi dan telusuri memori internal HP kamu.

5. Ubah format PDF
Terakhir kamu bisa mengubah nama dokumen sesuai dengan keinginan. Untuk cara membuat PDF, kamu tinggal ketuk menu dropdown pada pilihan format file dan pilih .pdf.
Jika sudah dirasa yakin, kamu tinggal ketuk Ekspor ke PDF dan tunggu hingga semua proses selesai.
