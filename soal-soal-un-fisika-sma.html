Pada sebuah peredaran darah hewan ,jari-jari pembuluh nadinya adalah 1,2 cm. Darah mengalir dari pembuluh nadi dengan kelajuan 0,4 m/s menuju ke semua pembuluh kapiler yang ada dengan kelajuan rata-rata 0,5 mm/s dan jari-jari pembuluh kapiler [tex]8\times 10^-^4[/tex] cm. Jumlah pembuluh kapiler adalah..miliyar. A. 2,1 B.1,8 C. 1,5 D. 1,2 E. 0,6 2. Dua perahu disungai bergerak lurus saling mendatangi.Laju perahu pertama dan kedua relatif terhadap tanah adalah v dan 2v. Ketika jarak kedua perahu 72 meter dan air sungai tidak mengalir, kedua perahu berpapasan setelah bergerak selama 20 detik.Jika jarak kedua perahu 72 meter dan air sungai mengalir searah perahu pertama dengan laju 1 m/s , kedua perahu berpapasan setelah bergerak selama.... detik A. 18 B. 19 C. 20 D. 21 E. 22


1.
diketahui :
R1 = 1,2 cm = 12x10⁻³ m
v1 = 0,4 m/s
v2 = 0,5 mm/s = 5x10⁻⁴ m/s
R2 = 8x10⁻⁴ cm = 8x10⁻⁶ m

ditanyakan :
n = ?

jawab :
n1 x v1 x R1² = n x v2 x R2²
1 x 0,4 x (12x10⁻³)² = n x 5x10⁻⁴ x (8x10⁻⁶)²
4x10⁻¹ x 144x10⁻⁶ = n x 5x10⁻⁴ x 64x10⁻¹²
576x10⁻⁷ = 320 x 10⁻¹⁶ n
n = 576x10⁻⁷/320x10⁻¹⁶
n = 1,8 x 10⁹
n = 1,8 milyar


2.
diketahui :
v1 = v
v2 = 2v
s = 72 m
t = 20 s

ditanyakan :
t berpapasan saat v arus 1 m/s = ?

jawab :
saat arus tenang
t1 = t2
s1/v1 = s2/v2
s1/v = s2/2v
s1 = s2/2v x v
s1 = 1/2 s2
s2 = 2 s1 (pers 1)

s1 + s2 = 72 m
s1 = 72 - s2

karena s2 = 2s1 maka
s2 = 2 (72 - s2)
s2 = 144 - 2s2
s2 + 2s2 = 144
3 s2 = 144
s2 = 48 m

s2 = v2 x t
48 = v2 x 20
v2 = 48/20 = 2,4 m/s

karena
v1 = 1/2 v2 maka,
v1 = 1/2 x 2,4 = 1,2 m/s

mencari t berpapasan saat v arus 1 m/s searah dengan v1
v1' = v arus + v1
v1' = 1 + 1,2 = 2,2 m/s

v2' = v2 - v arus
v2' = 2,4 - 1 = 1,4 m/s

mencari jarak berpapasan:;
t1 = t2
s1/v1' = s2/v2'
s1/2,2 = s2/1,4
s1 = 2,2/1,4  s2
s1 = 11/7  s2

s1 + s2 = 72
s1 = 72 - s2

s1 = s1
11s2/7 = 72 - s2
11 s2 = 7 (72 - s2)
11 s2 = 504 - 7 s2
18 s2 = 504
s2 = 28 m

mencari t berpapasan, karena nilai t sama maka dapat dipilih salah satu, saya memilih menggunakan s2
t = s2/v2'
t = 28/1,4
t = 20 s

jadi kedua perahu bergerak selama 20 s ketika berpapasan
