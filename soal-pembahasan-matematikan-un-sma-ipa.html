13. matematikan ujian sekolah​


Jawaban:

Untuk menyelesaikan masalah ini, kita harus menerapkan dua aturan dasar dalam komposisi fungsi dan fungsi inversi, yaitu:

Komposisi fungsi: (g o f)(x) = g(f(x))

Fungsi inversi: jika f(x) = y, maka f^-1(y) = x

Kita dapat menyelesaikan masalah ini dengan langkah-langkah berikut:

Kita harus menentukan fungsi f(x) dan g(x) terlebih dahulu, dengan menggunakan informasi yang diberikan dalam soal:

f^-1(x) = x + 3, maka f(x) = x^-1(x-3)

g^-1(x) = (x-4)/2, maka g(x) = 2x+4

Selanjutnya, kita harus mencari (g o f)(x), yaitu hasil dari komposisi fungsi g dengan f:

(g o f)(x) = g(f(x))

= g(x + 3)

= 2(x + 3) + 4

= 2x + 10

Kemudian, kita harus mencari invers dari (g o f)(x), yaitu (g o f)^-1(x):

Jika (g o f)(x) = y, maka (g o f)^-1(y) = x

Kita dapat menyelesaikan persamaan ini untuk mencari x:

y = 2x + 10

x = (y - 10) / 2

Maka, (g o f)^-1(x) = (x - 10) / 2

Jadi, fungsi (g o f)^-1(x) adalah (x - 10) / 2.

32. matematikan lgi perlu


Diberikan SPLTV berikut:
4/(x + 1) + 5/(y + 2) – 2/(z – 3) = 2
2/(x + 1) – 10/(y + 2) = -1
8/(x + 1) + 16/(z – 3) = 12
Misalkan:
a = 1/(x + 1)
b = 1/(y + 2)
c = 1/(z – 3)
Kita peroleh SPLTV baru:
4a + 5b – 2c = 2  (1)
2a – 10b = -1       (2)
8a + 16c = 12       (3)
Dari (2) kita dapatkan 2a = 10b – 1.
Substitusikan ke (1) dan (3), diperoleh:
2(10b – 1) + 5b – 2c = 2
↔ 20b – 2 + 5b – 2c = 2
↔ 25b – 2c = 4       (4)
4(10b – 1) + 16c = 12
↔ 40b – 4 + 16c = 12
↔ 40b + 16c = 16   (5)
Eliminasi c dari (4) dan (5).
25b – 2c = 4        |×8|  200b – 16c = 32
40b + 16c = 16    |×1|  40b + 16c = 16
                                      ------------------------  +
                                          240b   =  48
                                 ↔    b = 48/240 = 1/5
Substitusikan nilai b = 0,2 ke (2) dan (5)
2a – 10(1/5) = -1
↔ 2a = -1 + 2
↔ a = 1/2
40(1/5) + 16c = 16
↔ 16c = 16 – 8
↔ c = 8/16 = 1/2
Kembalikan nilai a, b, c ke pemisalan awal.
a = 1/(x + 1) = 1/2
↔ x + 1 = 2
↔ x = 1
b = 1/(y + 2) = 1/5
↔ y + 2 = 5
↔ y = 3
c = 1/(z – 3) = 1/2
↔ z – 3 = 2
↔ z = 5
Dapat kita hitung, nilai (x + 3y) : z = (1 + 3(3)) : 5 = (1 + 9) : 5 = 10 : 5 = 2
