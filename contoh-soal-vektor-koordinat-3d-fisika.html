Kelas : X SMA
Bab : Vektor

Pembahasan :
5d + 3a + 2b = 2c + 3d - 2b
2d + 3[-2 -8 6] + 2[5 4 -5] = 2[3 4 -2] - 2[5 4 -5]
2d + [-6 -24 18] + [10 8 -10] = [6 8 -4] - [10 8 -10]
2d + [4 -16 8] = [-4 0 6]
2d = [-4 0 6] - [4 -16 8]
2d = [-8 16 -2]
d = [-4 8 -1] = -4i + 8j - k

18. Diketahui vektor a = -2i -8j +6k , vektor b = 5i +4j -5k dan vektor c = 3i +4j -2k. Jika 5d +3a +2b = 2c +3d -2b , vektor d = ....


Jawab:

langsung masukkan vektor kedalam persamaan

5d +3a + 2b = 2c + 3d - 2b

5d - 3d = -3a - 4b + 2c

3d = -3 (-2i - 8j + 6k) - 4 (5i + 4j - 5k) + 2 (3i + 4j - 2k)

3d = (6i + 24j - 18k) - (20i + 16j - 20k) + (6i + 8j - 4k)

3d = -8i + 12j - 2k

d = -8/3 i + 4j - 2/3 k

Jawab:

a=-2i - 8j +6k

b= 5i + 4j - 5k

c= 3i +4j -2k

Ditanya:

5d + 3a +2b = 2c + 3d -2b

Jawab:

5d + 3a + 2b = 2c +3d -2b

5d - 3d = -3a - 2b - 2b + 2c

2d = -3a - 4b + 2c

2d = -3(-2i -8j + 6k) -4(5i + 4j -5k) + 2(3i + 4j - 2k)

2d = (6i + 24j - 18k) - (20i + 16j - 20k) + (6i + 8j -4k)

2d = (-8i + 16j -2k)

d= (-8i +16j -2k)/2

d= -4i + 8j -1k
