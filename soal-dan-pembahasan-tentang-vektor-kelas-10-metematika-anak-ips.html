19. TEMATIKA KELAS XHimpunan penyelesaiandari |2x - 4| = 3 adalah...​


Himpunan penyelesaian dari |2x-4|=3 adalah...

Nilai Mutlak

merupakan suatu jarak antara bilangan tertentu dengan nol pada garis bilangan real. Karena jarak, maka nilainya selalu positif (tidak ada yang negatif). Sehingga nilai mutlak yaitu nilai yang selalu positif.

untuk menyelesaikan soal dapat kita kerjakan dengan cara:

1) kita kuadratkan kedua ruas

|2x-4|=3

(2x - 4)² = 3²

4x² - 16x + 16 = 9

4x² - 16x + 16 - 9 = 0

4x² - 16x + 7 = 0

(2x - 7)(2x - 1) = 0

2x - 7 = 0

2x = 7

x = 7/2

atau

2x - 1 = 0

2x = 1

x = 1/2

2) dengan cara jika x < 0 dan jika x > 0

jika x > 0

|2x-4|=3

2x - 4 = 3

2x = 3 + 4

2x = 7

x = 7/2

jika x < 0

|2x-4|=3

-(2x - 4) = 3

-2x + 4 = 3

-2x = 3 - 4

-2x = -1

x = 1/2

jadi himpunan penyelesaiannya adalah {1/2 , 7/2}

Uₙ = (-8) - 4n + 4
Uₙ = -4n - 4

U₁₀₀ = -404

Uₙ = 3 + 6n - 6
Uₙ = 6n - 3

U₁₀₀ = 6(100) - 3
U₁₀₀ = 600 - 3
