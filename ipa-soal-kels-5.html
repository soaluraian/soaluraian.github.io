34. tolonggg cepatttt secepatttnyaaaa kels 8


Penjelasan dengan langkah-langkah:

Polanya :

1, 5, 13, 25, .... (aritmatika bertingkat)

a = 1 dan b = U2 - U1 = 5 - 1 = 4

c = (U3 - U2) - (U2 - U1)

c = (13 - 5) - (5 - 1)

c = 8 - 4

c = 4

Rumus pola ke-n :

Un = a + (n - 1).b + ½(n - 1)(n - 2).c

Un = 1 + (n - 1).4 + ½(n² - 3n + 2).4

Un = 4n - 4 + 1 + 2(n² - 3n + 2)

Un = 2n² - 6n + 4n - 3 + 4

Un = 2n² - 2n + 1

Maka, pola ke-51 adalah

U51 = 2(51)² - 2(51) + 1

U51 = 2(2.601) - 102 + 1

U51 = 5.202 - 101

U51 = 5.101
